# KENTERA 解説動画 台本生成（患者向け専用）

素材テキストを貼り付けるだけで、患者向け解説動画の**3種ファイルを同時生成・自動保存**します。

---

## 使い方（2ステップ）

1. **Cursor のチャットに素材テキストを貼り付ける**
2. **「台本作って」と依頼する**

→ 以下の3ファイルが自動で生成・保存されます。

---

## 生成されるファイル（3種）

| ファイル | 保存先 | 用途 |
|--------|-------|-----|
| `{タイトル}_slide_{日時}.md` | `出力/スライド/` | スライド作成ツールに流し込む用 |
| `{タイトル}_script_{日時}.md` | `出力/読み上げ台本/` | AI音声・実際の読み上げ用（テキストのみ） |
| `{タイトル}_summary_{日時}.md` | `出力/要約/` | 内容保管・共有用（箇条書き要約） |

---

## フォルダ構成

```
KENTERA一般台本/
├── 出力/
│   ├── スライド/          # ① スライド生成用（自動保存先）
│   ├── 読み上げ台本/      # ② 読み上げテキスト用（自動保存先）
│   └── 要約/             # ③ テキスト要約用（自動保存先）
├── rules/
│   ├── common.md         # 共通ルール
│   ├── format.md         # 3種フォーマット定義
│   ├── expressions.md    # 表現バリエーション集
│   └── patient.md        # 患者向けルール
├── .cursor/rules/        # Cursor 自動実行ルール
└── README.md
```
