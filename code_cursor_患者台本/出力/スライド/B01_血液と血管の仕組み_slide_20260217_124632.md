# スライド6: ポイント① 5μmと8μmのパラドックス
## ナレーション
はい、ここから本編です。1つ目の「血管より赤血球のほうが大きい」というお話です。ここで大事な数字が2つあります。「5」と「8」です。
何の数字だと思いますか？正解は、「血管の太さ」と「赤血球の大きさ」です。
全身の細胞に酸素を届ける毛細血管、この太さは約5マイクロメートル。
対して、酸素を運ぶトラック役の赤血球は、約8マイクロメートルなんです。おかしいと思いませんか？道幅より、トラックのほうが大きいんです。
物理的に考えたら、絶対に通れないはずですよね。でもここに、私たちの体のすごい秘密が隠されているんです。

# スライド7: ポイント①続き 赤血球の変形能力
## ナレーション
でも、私たちの体はこれをちゃんと通過させています。どうやっているか。答えは「変形」です。
健康な赤血球は、お餅みたいに柔らかいんですよ。
5マイクロメートルの狭い道を、8マイクロメートルの赤血球が体を「ニョーン」とねじらせて、無理やりくぐり抜けていく。
奈良の大仏さまの鼻の穴くぐりみたいな感じですね。ここが大事なんですが、実はこの「無理やり通る」ことに意味があるんです。
赤血球は血管の壁にこすりつけられてギュッと絞られることで、初めて酸素や栄養素を細胞に手渡せる構造になっています。
つまり、スカスカに通れる広い道では、酸素は届けられないんです。
面白い仕組みですよね。

# スライド8: ポイント② 血流が悪い＝赤血球が硬い
## ナレーション
では、2つ目の「血液サラサラの正体」についてです。よく「血液ドロドロ」って言いますよね。でも実は、液体が濃いとかベタベタしている、という話ではないんです。
本当の問題は2つ。
1つは、赤血球がカチコチに硬くなって変形できないこと。
もう1つは、血管自体が硬くなって柔軟性を失っていること。赤血球が硬くなると、さっきの狭いトンネルに入ろうとした瞬間に、入り口で渋滞を起こしてしまいます。
これが「詰まる」の正体なんですね。でも裏を返せば、赤血球の柔らかさを取り戻せれば、流れも良くなるということなんです。

---
# スライド9: ポイント②続き ゴースト血管と冷え性
## ナレーション
渋滞が起きると、末端の細胞に酸素が届かなくなりますよね。酸素が届かないと、熱も作れません。これが「冷え性」の原因の一つです。
さらにひどくなると、使われなくなった毛細血管が消えてしまう。
これを「ゴースト血管」と呼びます。ちょっとドキッとする名前ですよね。体温が1度下がると、免疫力は約30％下がると言われています。
代謝にも影響が出てきます。でも、ここで希望のある話をしますね。これらはすべて、あの「5と8の通過儀礼」がスムーズにいけば改善が期待できるんです。

---
# スライド10: ポイント③ 高血圧の正体
## ナレーション
3つ目は「高血圧やコレステロールは防衛反応」というお話です。これが一番大事かもしれません。心臓はポンプですが、ただ闇雲に血液を送っているわけではありません。
実は体は、約10秒に1回、全身の血管と連絡を取り合っているんです。
「おーい、足の先までちゃんと届いたか？」って確認してるんですね。もし末端から「届いてません！」という返事が来たら、心臓はどうするか。
圧力を上げて、無理やり届けようとします。
これが「高血圧」の正体です。つまり高血圧は、心臓が悪いわけじゃないんです。
「末端に届かないから、その結果として圧を上げている」という、体の防衛反応なんですね。むしろ、体が一生懸命あなたを守ってくれている証拠でもあるんですよ。

# スライド11: ポイント③続き コレステロールの正体
## ナレーション
そしてもう一つ。高い圧力で血液を流し続けると、当然、血管は傷つきます。傷ついた血管が破裂したら大変ですよね。
そこで登場するのが「コレステロール」です。
「悪玉」なんて呼ばれて嫌われていますけど、実は彼らの仕事は「血管の修復パテ」、つまり接着剤なんですよ。血管が傷ついているから、補修するために集まってきているだけ。
だからコレステロール値が高いということは、「体のあちこちで血管が傷ついて、今まさに修復工事中ですよ」というサインなんです。
だから、数値だけを見るのではなく、「なぜ修復が必要になっているのか」に目を向けると、もっと前向きな対策が見えてくるんです。

# スライド13: 肩こりの真実
## ナレーション
ちなみに、マッサージに行っても治らない慢性的な肩こり、ありませんか？実はこれにも、血圧の仕組みが関係しています。
血圧を上げて無理やり血液を流すとき、体は血管が破裂しないように、周りの筋肉をギュッと固めてガードするんです。
つまり、体が命を守るために、あえて筋肉を硬くしている。だから無理にほぐしても、すぐに元に戻ってしまうんですね。でも逆に言えば、血流が改善すれば、体が自然とガードを解いてくれる。
つまり、根本からラクになれる可能性があるということなんです。
