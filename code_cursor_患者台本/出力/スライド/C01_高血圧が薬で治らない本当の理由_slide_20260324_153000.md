# 3行サマリー
- この動画のゴール：高血圧の薬を飲んでも「治癒」しない理由を理解し、根本から体を変えるための正しい優先順位がわかる
- 視聴後にできること：「寛解」と「治癒」の違いを家族に説明でき、体質改善の第一歩として測定を決断できる
- 次の一手（CTA）：測定で自分の体の「答え合わせ」をする

---

# スライド1: 「治る」には2種類ある
## ナレーション
こんにちは！今回のテーマは「高血圧が薬で治らない本当の理由」です。
高血圧のお薬を何年も飲んでいるのに、「結局いつまで飲み続ければいいの？」と疑問に感じたこと、ありませんか？
実はここに大きな落とし穴があるんです。「治る」という言葉には2種類の意味があって、薬で言う「治る」は、皆さんが思っている「治る」とは別物なんです。
今日はこの違いを知るだけで、「何をどの順番でやればいいか」がクリアに見えてきますよ。一緒に見ていきましょうね。

---

# スライド2: 今日おさえる3つのポイント
## ナレーション
結論を先にお伝えしますね。今日覚えていただきたいのは3つです。
1つ目、薬の「治る」は一時しのぎの「寛解」。根本から良くなる「治癒」とは別物です。
2つ目、治癒への最短ルートは「骨格→内臓→栄養」の順番を守ること。
3つ目、迷いをなくすカギは、自分の体を「測定」して原因を確定させること。
では、さっそく本題にいってみましょう！

# スライド3: 寛解と治癒の違い
## ナレーション
1つ目の「薬の"治る"は一時しのぎ」というお話です。
まず知っていただきたい言葉が2つあります。「寛解」と「治癒」です。
寛解というのは、薬で一時的に症状が治まっているけれど、薬をやめればまた戻ってしまう状態のこと。
一方の治癒というのは、薬がなくても健康な状態がずっと続くことです。
これ、家の雨漏りに例えるとすごくわかりやすいんですよ。
雨漏りしている家で、床が濡れないようにバケツを置く。これが「寛解」です。水は受け止められるけど、雨漏りそのものは止まっていませんよね。
一方、屋根に登って穴をふさぐ。これが「治癒」です。屋根を直せば、バケツはもう必要なくなる。
皆さんが「高血圧を治してほしい」と病院に行くとき、求めているのは当然「屋根の修理」、つまり治癒ですよね。

# スライド4: 「治りますよ」のすれ違い
## ナレーション
ところが、高血圧を根本から「治癒」できる薬は、まだ見つかっていません。
だから病院では、ガイドラインに沿って、一時的に数値を下げる「寛解」のための薬、つまりバケツが処方されるんですね。
ここで問題なのが、法律上は寛解のことも治癒のこともまとめて「治る」と表現してよいことになっている点です。
お医者さまは「バケツで水を受けますよ」という意味で「治りますよ」と言っているのに、患者さんは「屋根を直してくれるんだ」と受け取ってしまう。
このギャップが、「何年飲んでもやめられない」という悩みの正体だったんです。
でもね、ここからが希望のある話ですよ。屋根の直し方、つまり治癒に向かうための道筋がわかれば、体はちゃんと変わっていけるんです。

---

# スライド5: 屋根を直す正しい手順
## ナレーション
では、屋根はどうやって直せばいいのか。ここが2つ目のポイントです。
実は、屋根を直すにも正しい手順があるんです。
体質改善の優先順位は、①骨格を整える、②内臓の機能を戻す、③栄養を入れる、この順番です。
どんなに良い屋根材を買ってきても、骨組みがグラグラだったら乗せられませんよね。まず骨組みを直して、下地を整えて、それから初めて屋根材を張る。体もまったく同じなんです。
だから、バケツ（薬）だけでは屋根は直らなかったんですね。バケツは雨水を受けてくれる大切な存在だけど、屋根の修理とは別の作業なんです。

---

# スライド6: 骨格→内臓→栄養の理由
## ナレーション
もう少し具体的にお話ししますね。ここ、知ると世界の見え方が変わるポイントです。
姿勢が崩れて骨格が歪んでいると、その中にある血管が圧迫されてしまいます。血管が圧迫されると、体は無理やり血液を押し流そうとして、血圧を上げるんです。
つまり、骨格の歪みが高血圧の原因の一つだった。冒頭で「薬を飲んでも治らない」とお話ししましたが、その理由がここに繋がっていたんですね。
さらに、姿勢が崩れると内臓も圧迫されます。ちょっと想像してみてください。ご飯を食べる前に、胃を3時間ギューッと押さえつけられたらどうですか？ とても消化なんてできませんよね。
内臓が圧迫されたままだと、せっかく良い栄養を摂っても吸収できない。だからバケツ（薬）だけ増やしても、屋根（根本原因）は直らなかったんです。
この土台作りを飛ばして、例えば最新の再生医療をいきなり受けると、1回130万円以上かかることもあります。
でも、この仕組みがわかった方はもう一歩リードしていますよ。正しい順番で骨格から整えていけば、体はちゃんと応えてくれます。これが一番コスパ良く治癒に向かう最短ルートなんです。

# スライド7: サプリ90万円が効かなかった理由
## ナレーション
さあ、3つ目のポイントにいきますよ。「測定で原因を確定させる」というお話です。
骨格と内臓を整えたら、いよいよ栄養療法の出番ですよね。でもここで、多くの方がつまずく落とし穴があるんです。
ある女性のケースをご紹介します。
「自分には亜鉛が足りない」と思い込んで、5年間で約90万円分のサプリを飲み続けた。なのに、数値がまったく改善しなかったんです。
なぜだと思いますか？

# スライド8: 有害金属という見えない壁
## ナレーション
答えは、体の中に溜まっていた「有害金属」が、ミネラルの吸収をブロックしていたからです。
いくら栄養を入れても、邪魔者がいたら届かない。屋根を直そうとしても、ガレキが乗ったままでは作業ができないのと同じです。
つまり、有害金属がいる限り、いくらサプリを飲んでも栄養は届かない。これも「薬を飲んでも治らない」の正体の一つだったんですね。
実は、日本人の慢性的な不調の60〜70％に、この有害金属が関わっていると言われています。
少し驚くかもしれませんが、ここに希望のタネが隠れています。邪魔者の正体がわかれば、取り除く方法もあるということですからね。
だからこそ必要なのが「測定」です。
病院で「異常なし」と言われて終わるのではなく、細胞レベルで「何が足りないのか」、そして「何が邪魔をしているのか」を数値で確定させる。
カーナビを想像してみてください。目的地を入れる前にまず「現在地」を確認しますよね。現在地がわからなければ、どんなに優秀なナビでもルートは出せません。測定は、まさにその「現在地の確認」なんです。

# スライド9: 今の薬はやめていいの？

ここまでの話を聞いて、「じゃあ今飲んでいる薬は意味がないの？」と不安に思った方もいるかもしれません。
安心してくださいね。この動画はお薬や医療を否定するものではありません。
雨漏りの話を思い出してください。屋根を直している最中に、バケツを外したらどうなりますか？ 床がびしょ濡れになりますよね。
だから、バケツ（薬）は屋根を直し終わるまで絶対に外してはダメなんです。お医者さまが処方されたお薬は、きちんと飲み続けてくださいね。勝手にやめちゃダメですよ〜。
今日お伝えしたかったのは、バケツで水を受けながら、同時に屋根の修理も進めていきましょう、ということなんです。これが一番賢いやり方ですからね。
気になることがあれば、いつでもお医者さまや私たちに気軽に相談してくださいね。

# スライド12: あなたの「現在地」を確認しませんか？
## ナレーション
最後に1つだけお伝えしますね。ここまで高血圧と体質改善の仕組みをお話ししてきましたが、実は「自分の体のどこで一番エラーが起きているか」は、一人ひとりまったく違うんです。骨格の歪み方も、有害金属の溜まり方も、栄養の過不足も人それぞれ。
今日お話しした内容は、まさに「測定」で原因がはっきりわかることと直結しています。だからこそ、「じゃあ、私の場合はどうなっているんだろう？」と気になった方には、測定での答え合わせをぜひ強くお勧めします。
せっかく治癒に向かう仕組みを理解できたのに、自分の体に合わない自己流の対策をして遠回りしてしまうのは、本当にもったいないですからね。
皆さんの体は、良くなる力をちゃんと持っています。あなたの体質に合わせた最短ルートを一緒に見つけていきましょうね。
