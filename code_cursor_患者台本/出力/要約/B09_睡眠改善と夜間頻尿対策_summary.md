【B09：睡眠改善と夜間頻尿対策】

〜知っておくと読みやすい用語〜
・膀胱（ぼうこう）→ 腎臓で作られた尿を一時的にためておく袋状の臓器。本来は風船のように柔らかい
・夜間頻尿（やかんひんにょう）→ 夜中に1回以上トイレで目が覚めてしまう症状
・夜間多尿（やかんたにょう）→ 夜間に体が尿を作りすぎてしまう状態。昼間のむくみが主な原因
・蓄尿障害（ちくにょうしょうがい）→ 膀胱が硬くなって十分に尿をためられなくなること
・クエン酸 → レモンや梅干しに含まれるすっぱい成分。血流改善や組織を柔らかくする働きがある
・メラトニン → 「睡眠ホルモン」と呼ばれる物質。暗くなると分泌され、眠気を誘う。昼間に日光を浴びると夜の分泌が増える
・利尿作用（りにょうさよう）→ 尿の量を増やす働きのこと。カフェインやアルコールに強い利尿作用がある

〜この動画でわかること〜
夜中にトイレで目が覚める原因は「老化」ではない。膀胱の硬化とむくみの仕組み、そしてクエン酸を使ったシンプルな対策がわかります。


【今日の3つのポイント】

【1. 夜中のトイレは「老化」ではなく、膀胱の「硬化」と昼間の「むくみ」が原因】
（例）ラグビーボールは元々牛の膀胱で作られていた。本来、膀胱は風船のように柔らかく伸び縮みする臓器。でも加齢や血流不足によって古くなったゴムホースみたいに「硬く」なってしまうことがある
・膀胱が硬くなると、少し尿が溜まっただけで壁がうまく伸びず、センサーが過敏に反応。実際はまだ溜められるのに「もう満タンだ！すぐ出して！」と脳に誤報を送ってしまう
（例）昼間、立ったり座ったりしているうちに重力で足にどんどん水分が溜まる（むくみ）。夜横になると重力から解放され、足に溜まっていた水分が一気に上半身や腎臓に戻る。体は「水が多すぎる！」と判断して夜中にせっせと尿を作ってしまう。つまり昼間のむくみを放置したまま寝ることは、夜中にトイレのアラームを自分でセットしているようなもの
・夜間頻尿がある方はそうでない方に比べて死亡率が約2倍というデータもある。睡眠が途切れること自体が体の修復を中断し、暗い中を歩くことで転倒→骨折→寝たきりにつながるリスクもある

【2. 夕方にクエン酸を飲んで、寝る前までに余分な水分を出し切る】
・ドラッグストアで数百円で買える「食用クエン酸」が、実は睡眠を守る2つの働きを持っている
　① 血液の流れを良くして、昼間のうちにしっかり水分を排出させる
　② 硬くなった膀胱のような組織を柔らかくする方向に働く
・飲むタイミングが最大のポイント。おすすめは「夕方4時〜5時ごろ」
（例）夕方にクエン酸を飲んで血液の巡りを良くし、寝る前までに体の中の余分な水分を「出し切ってしまう」。そうすると寝ている間に作られる尿の量が減って、朝までぐっすり眠れるようになる。寝る直前に飲むと逆にトイレが近くなるのでNG

【3. 寝る前のアルコールとカフェインは、夜間頻尿を悪化させる最大の落とし穴】
（例）アルコール、特にビールは飲んだ水分の10倍の速さで尿を作ると言われている。寝る前にビールを飲むのは、せっかく夕方に出し切った水分をもう一度体に入れ直しているようなもの。さらにアルコールは睡眠の質そのものも下げる。寝つきは良くなったように感じても眠りが浅くなって回復力が落ちる
・意外な盲点が「緑茶」。健康に良いイメージがあるが、コーヒー以上に利尿作用が強い場合もある。コーヒーも緑茶も昼間に飲んで老廃物を出すために使い、夜はノンカフェインの水やハーブティーに切り替える
・日中に少しでも日光を浴びると、睡眠ホルモン「メラトニン」の準備スイッチが入る


【今日からやること】

・今日 → ドラッグストアやスーパーで「食用クエン酸」を手に入れる。粉末タイプで数百円
・3日以内 → 夕方4〜5時にクエン酸を水に溶かして飲む習慣を開始（小さじ半分くらいから）。寝る前のビール・コーヒー・緑茶を少し控えめにしてみる
・1週間後 → 夜中にトイレで目覚める回数が変わったか振り返ってみる。「あれ、朝まで眠れた！」と感じたら体がちゃんと応えてくれている証拠
