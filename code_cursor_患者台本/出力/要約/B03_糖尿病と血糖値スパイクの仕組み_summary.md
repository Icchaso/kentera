【B03：糖尿病と血糖値スパイクの仕組み】

〜知っておくと読みやすい用語〜
・血糖値（けっとうち）→ 血液の中にどれくらい糖（ブドウ糖）が含まれているかを示す数値
・インスリン → すい臓から出るホルモン。血液中の糖を細胞に取り込ませて血糖値を下げる役割
・グルカゴン → すい臓から出るもう一つのホルモン。血糖値を上げる役割。インスリンとセットで血糖値を調整している
・血糖値スパイク → 食後に血糖値が急上昇→急降下すること。血管を傷つけ、眠気やだるさの原因になる
・動脈硬化（どうみゃくこうか）→ 血管の壁が硬く厚くなって、血液の流れが悪くなった状態

〜この動画でわかること〜
食後の眠気や高血糖が起きる"本当の理由"と、今日からできるシンプルな対策がわかります。


【今日の3つのポイント】

【1. 食後の眠気は「血糖値スパイク」のサイン】
・精製された糖質（白いごはん、菓子パン、甘いジュースなど）を一気に食べると、血糖値が「ドーン！」と急上昇する
・体は「大変だ、血管が砂糖漬けになる！」と慌ててインスリンを大量に出す → 血糖値が「ドーン！」と急降下
（例）この急上昇→急降下はまるでジェットコースター。これが「血糖値スパイク」。食後の強烈な眠気やだるさ、イライラの正体はこのジェットコースターであり、「やる気の問題」でも「性格の問題」でもない
・血糖値の急激な上下動は血管の内側をガリガリと傷つけ、動脈硬化の大きな原因にもなる

【2. 血糖値が下がらないのは「上げるホルモンの暴走」も関係している】
・血糖値を下げる「インスリン」だけでなく、上げるホルモン「グルカゴン」もある
・本来、食事をして血糖値が上がったら「もう糖分はいらないよ」とグルカゴンの分泌は止まる
・ところが糖尿病予備軍の一部では、食後なのにグルカゴン（上げるほう）が出続けてしまう
（例）車のアクセルとブレーキを同時に踏んでいる状態。ブレーキ役のインスリンがいくら頑張っても、アクセル役のグルカゴンが踏み込まれていたら血糖値は下がらない。体はどんどん疲弊してしまう

【3. 運動の目的は「カロリー消費」ではなく「食後の血糖値を抑えること」】
・運動で消費できるカロリーはびっくりするほど少ない。おにぎり1個分のカロリーを消費するのに何十分も走らなきゃいけない
・筋肉を動かすと、インスリンの助けを借りなくても、筋肉が直接血液中の糖を取り込んでくれる
・おすすめは「食後1時間」。ちょうど血糖値がピークを迎えようとするタイミングで散歩やスクワットをする
（例）上がりすぎようとする血糖値の山を、筋肉の力で物理的に低く押さえつけるイメージ。山が低くなればインスリンを大量に出す必要もなくなり、急降下も起きず、食後の眠気も防げる


【今日からやること】

・今日 → 次の食事のあと、「食後1時間」を意識して5分だけ歩いてみる。階段の上り下りだけでもOK
・3日以内 → 食事の順番を変える。ごはんやパンの前に、野菜やおかずを先に食べる。これだけで血糖値の急上昇がゆるやかになることがわかっている
・1週間後 → 食後の眠気がどう変わったか振り返ってみる。「あれ、前より楽かも」と感じたら、体がちゃんと応えてくれている証拠


〜ひとことメッセージ〜
食後の眠気は意志の弱さではなく、体の中で起きている「仕組み」のサイン。食後1時間にちょっと動くだけで、体はちゃんと応えてくれます。
