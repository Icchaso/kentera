【B06：コレステロールと高血圧の総論】

〜知っておくと読みやすい用語〜
・LDLコレステロール → メディアで「悪玉」と呼ばれる血液中の脂質。実際は傷ついた血管を修復する役割がある
・HDLコレステロール → 「善玉」と呼ばれるほう。余分なコレステロールを回収する役割
・ナトリウム → 塩（塩化ナトリウム）の主成分。体内の水分量や血圧を調整している
・カリウム → 野菜や果物に多いミネラル。ナトリウムとバランスを取って血圧を安定させる役割
・動脈硬化（どうみゃくこうか）→ 血管の壁が硬く厚くなり、血液の流れが悪くなった状態

〜この動画でわかること〜
コレステロールや血圧の数値は「下げるもの」ではなく「読みとくもの」。体からの手紙の読み方がわかります。


【今日の3つのポイント】

【1. 悪玉コレステロールは「血管の修復パテ」】
・LDL（悪玉）コレステロールの「悪玉」はメディアがつけた名前であり、医学用語ではない
・本当の役割は、傷ついた血管の壁を塞ぐ「修復パテ（補修材）」
（例）火事の現場に消防車がたくさんいると「あの場所は危ない」と思うけど、消防車が悪者ではない。火を消しに来てくれている。コレステロールも同じ。体を助けに来ている「味方」
・コレステロール値が高い ＝ 体のあちこちで血管が傷ついて、今まさに修復工事中というサイン
・もし薬で修復材（コレステロール）だけを減らしたら → 工事は中断され、血管は傷ついたまま放置される
・下げるべきは数値ではなく「血管を傷つけている炎症の原因」。炎症の原因を整えれば、体は自然と修復を終えてコレステロールを減らしていく

【2. 高血圧は、末端に血液を届けるための防衛反応】
・体には約100億本以上の血管があるが、心臓というポンプはたった1つ
・心臓は10秒に1回、全身の細胞と連絡を取り合って「酸素届いたか？」と確認している
・末端から「酸素が届いていません！」と返事が来ると、脳が「圧力を上げて、なんとか届けろ！」と命令 → これが高血圧の正体
・高血圧は末端の細胞を生かすための防衛反応。無理に下げると末端が酸欠になるリスクがある
（例）血圧が高い状態が続くと血管破裂のリスクがあるので、体は血管が破裂しないよう周りの筋肉をギュッと固めてガードする。これが「マッサージに行ってもすぐ戻る頑固な肩こり」の正体。肩こりは単なるコリではなく血管を守るための鎧（よろい）。血管の状態が良くなれば体は鎧を脱ぐ必要がなくなり、肩こりも自然と楽になる

【3. 減塩よりも「カリウム」をしっかり摂ること】
・血圧は「ナトリウムとカリウムの比率」で決まる
（例）シーソーの両端にナトリウムとカリウムが乗っているイメージ。現代の食生活は加工食品が増えて野菜が減った結果、カリウムが圧倒的に不足。だから相対的にナトリウムが多くなって血圧が上がったりむくんだりする
・対策は味気ない減塩ではなく、新鮮な野菜（カリウム）をしっかり摂ること
・さらにドロドロ血の原因になる糖質や酸化した油を控える。血管というホースがきれいになれば心臓は無理に圧を上げる必要がなくなり、自然と血圧は落ち着いてくる


【今日からやること】

・今日 → 食事に緑の野菜を1品プラス。ほうれん草、小松菜、ブロッコリー…何でもOK。それだけでカリウムの補給が始まる
・3日以内 → 健康診断の結果を「下げなきゃ」ではなく「体は何を伝えようとしているんだろう？」の目線で読み直してみる
・1週間後 → 加工食品やインスタント食品をほんの少し減らして、そのぶん生の野菜や自炊を増やしてみる。「へぇ」と思ったことはぜひご家族にも話してみてください。アウトプットすると定着する


〜ひとことメッセージ〜
数値は敵ではなく、体からの「手紙」。その手紙を破り捨てるのではなく、何が書いてあるのかを読んであげてください。
