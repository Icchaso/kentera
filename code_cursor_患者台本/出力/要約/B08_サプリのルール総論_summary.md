【B08：サプリのルール総論】

〜知っておくと読みやすい用語〜
・イオンチャネル → 細胞の表面にある極小の穴（約0.7ナノメートル）。栄養はこの穴を通って細胞に入る
・ナノメートル → 1mmの100万分の1。目に見えないどころか、想像もつかないくらい小さい単位
・ナノ化・イオン化 → 栄養素の粒を極小サイズに加工して、細胞の穴を通れるようにすること
・生体利用率（バイオアベイラビリティ）→ 口から摂った栄養のうち、実際に体が使える割合のこと
・自律神経（じりつしんけい）→ 心臓や胃腸など、自分の意志では動かせない臓器をコントロールしている神経。ストレスの影響を受けやすい

〜この動画でわかること〜
サプリ選びで失敗しないための3つの判断軸（選び方・やめ時・飲み方）がまとめてわかります。


【今日の3つのポイント】

【1. 「量（mg）」ではなく「サイズ」で選ぶ】
・細胞には栄養を取り込む穴「イオンチャネル」があり、そのサイズは約0.7ナノメートル。目に見えないどころか、想像もつかないくらい極小の穴
（例）この穴を「ゴルフのカップ」だと思ってください。市販の安価なサプリや質の良くないミネラルは粒子がこの穴に対して大きすぎる。つまり「ボウリングの玉」。ゴルフのカップにボウリングの玉を入れようとしても絶対に入らない。穴の上を転がって素通りするだけ
・パッケージの「1000mg配合！」は、言い換えれば「ボウリングの玉を100個詰めました！」と言っているようなもの。穴に入らなければ全部そのまま体の外に出ていく
・大事なのは「ナノ化・イオン化されていて細胞の穴を通れるかどうか」（＝生体利用率・バイオアベイラビリティ）
・覚えることは1つ：「量ではなくサイズで選ぶ」

【2. 「ビタミンは短期戦、ミネラルは長期戦」】
・ビタミン（B群・Cなど）＝ 体の中で代謝に使われる「潤滑油」。吸収も早いが使われるのも早い。1〜2週間で体内濃度が変わる → 風邪や疲れに短期集中で効果的
・ミネラル（カルシウム・亜鉛・マグネシウムなど）＝ 骨や細胞そのものの「材料」。定着に半年〜数年かかる
（例）ミネラルは「家の基礎工事」。基礎が完成するまでにはどうしても時間がかかる。「1ヶ月で効果がない」と言ってやめるのは、基礎工事が終わっていないのに「全然家が建たない！」と言っているようなもの
・時間軸を知っておくだけで、やめ時を間違えなくなる

【3. 飲むタイミングと気持ちの持ちようで吸収率は変わる】
・タイミング：夜10時〜朝7時は体が「排泄（デトックス）」モード。朝イチよりも日中〜夕方のほうが効率よく吸収できる
・気持ち：「本当に効くのかな…」と疑いながら飲むと、ストレスで胃酸の分泌や腸の動きが鈍くなり、実際に吸収率が下がることがある
（例）逆に「これで細胞が元気になっていくぞ」と前向きな気持ちで飲むと、体は受け入れ態勢になる。胃や腸は自律神経の影響を強く受けるので、精神論ではなく体の仕組みとして理にかなった話
・細かい数字を気にしてストレスを溜めるより、良いイメージで飲むほうが効果的


【今日からやること】

・今日 → 今飲んでいるサプリがあれば裏面の表示を見てみる。「何mgか」ではなく「吸収されやすい形になっているかな？」という視点で眺めてみる
・3日以内 → 自分の悩みが「短期的な疲れやだるさ（ビタミン向き）」か「長期的な体質改善（ミネラル向き）」か考えてみる。紙に書き出すとスッキリする
・1週間後 → サプリを飲むとき「これで体が整っていくぞ」と良いイメージを持って飲んでみる。疑いながら飲むよりもずっと体のためになる


〜ひとことメッセージ〜
迷ったら「量よりサイズ」。それだけ覚えておけば、サプリ選びで振り回されることはなくなります。
