【B02：現代食の落とし穴と整える優先順位】

〜知っておくと読みやすい用語〜
・3大栄養素 → タンパク質・脂質・炭水化物。体のエネルギーや材料になるもの
・5大栄養素 → 3大栄養素にビタミン・ミネラルを加えたもの
・ミネラル → カルシウム・鉄・亜鉛・マグネシウムなど。体の調整役で、骨や細胞の材料にもなる
・新型栄養失調 → カロリーは十分なのに、ビタミンやミネラルが不足している現代特有の栄養失調
・イオンチャネル → 細胞の表面にある極小の穴。栄養素はこの穴を通って細胞の中に入る
・イオン化 → 栄養素の粒をとても小さくして、細胞の穴を通れる状態にすること

〜この動画でわかること〜
「しっかり食べているのに調子が悪い」の正体が、車の例えでスッキリわかります。


【今日の3つのポイント】

【1. 体は「ガソリン」だけでは動かない。「オイル」が必要】
・栄養には2つのグループがある
　タンパク質・脂質・炭水化物（3大栄養素）＝ 車の「ガソリン」と「車体」
　ビタミン・ミネラル ＝ 「エンジンオイル」や「潤滑油」
（例）ガソリン満タンでもオイル交換をまったくしていない車を想像してみてください。エンジンは焼き付いて、煙を上げて故障しますよね。これが今の私たちの体で起きていること
・コンビニやスーパーの食事はカロリー（ガソリン）はたっぷりだが、ビタミン・ミネラル（オイル）が圧倒的に不足 → 「新型栄養失調」と呼ばれている
・カロリーは足りすぎているのに、それを体の中で使うための栄養が空っぽという状態
・足りていないオイルを補えばいい。やるべきことがはっきりしているぶん、改善の道筋は見えやすい

【2. 慢性的な不調の原因は、ほとんどが「ミネラル不足」】
・ビタミンとミネラルでは、まず「ミネラル」を優先すべき
・ビタミンは体内での代謝が早く、1〜2週間で変動する「短期戦」。風邪のときにビタミンCを短期で摂ると効くのはこの仕組み
・ミネラルは骨や細胞の一部として組み込まれるため、入れ替わるのに半年〜数年かかる「長期戦」
（例）ミネラルは「家の基礎工事」のようなもの。数ヶ月、年単位でコツコツ積み上げて、初めて体質という土台が変わっていく。「サプリを1ヶ月飲んだけど効果がないからやめた」は、基礎工事が終わっていないのに「家が建たない！」と言っているようなもの

【3. 「量」より「サイズ」。細胞に入れるものを選ぶ】
・細胞には栄養を取り込む小さな穴（イオンチャネル、約0.7ナノメートル）がある
・質の悪いサプリや加工されすぎた食品のミネラルは、この穴に対して粒子が大きすぎる
（例）ゴルフのカップの穴に、ボウリングの玉を入れようとしているイメージ。どう頑張っても入らない。「1000mg配合！」はボウリングの玉を100個詰めたようなもの。穴に入らなければ全部そのまま体の外に出ていくだけ
・大事なのは含有量ではなく、「細胞の穴を通れるサイズにイオン化されているかどうか」


【今日からやること】

・今日 → 食事のとき「この中にオイル（ビタミン・ミネラル）は入ってるかな？」と一瞬だけ意識する。それだけでOK
・3日以内 → 海藻・出汁・ごま・ナッツなど、ミネラル豊富な食材を1品プラス。味噌汁にわかめを入れる、ごはんにごまをかける、それだけで十分
・1週間後 → サプリを使っている方は裏面の表示を見て「この成分は吸収されやすい形かな？」と意識してみる


〜ひとことメッセージ〜
足りていない「オイル」を補えばいい。やるべきことがはっきりしているぶん、改善の道筋は見えやすいです。
