【B01：血液と血管の仕組み】

〜知っておくと読みやすい用語〜
・赤血球（せっけっきゅう）→ 血液の中にある赤い粒。全身に酸素を届けるトラック役
・毛細血管（もうさいけっかん）→ 体中に張りめぐらされた、髪の毛より細い血管。酸素の受け渡しはここで行われる
・マイクロメートル → 1mmの1000分の1。肉眼では見えないサイズ
・コレステロール → 血液中にある脂質の一種。「悪玉（LDL）」「善玉（HDL）」と呼ばれるが、どちらも体に必要な役割がある
・ゴースト血管 → 血流が悪くなり、使われなくなって消えてしまった毛細血管のこと
・クエン酸 → レモンや梅干しに含まれるすっぱい成分。血流や赤血球の柔軟性に関わる

〜この動画でわかること〜
血圧やコレステロールの数値が上がる"本当の理由"が、たった1つの仕組みからスッキリわかります。


【今日の3つのポイント】

【1. 血管より赤血球のほうが大きい】
・毛細血管の太さは約5マイクロメートル、赤血球の大きさは約8マイクロメートル
・道幅よりトラックのほうが大きいのに、赤血球はお餅のように柔らかく変形して通り抜ける
（例）奈良の大仏さまの鼻の穴くぐりのように、8マイクロメートルの赤血球が体を「ニョーン」とねじらせて5マイクロメートルの道を無理やりくぐり抜ける
・この「ギュッと絞られる」仕組みがあるからこそ、赤血球は血管の壁にこすりつけられて酸素や栄養を細胞に手渡せる。スカスカに通れる広い道では、実は酸素は届けられない

【2. 「血液サラサラ」の正体は、赤血球が柔らかく変形できること】
・「血液ドロドロ」とは、液体が濃いとかベタベタしているという話ではない
・本当の問題は、赤血球がカチコチに硬くなって変形できないこと、そして血管自体が硬くなっていること
（例）狭いトンネルの入り口に硬い赤血球が次々と詰まって渋滞を起こしているイメージ。これが「詰まる」の正体
・末端に酸素が届かなくなると熱も作れない → 冷え性の原因。さらにひどくなると使われなくなった毛細血管が消える「ゴースト血管」に
・体温が1度下がると免疫力は約30%低下し、代謝にも影響が出る
・でも赤血球の柔らかさを取り戻せば、流れは改善できる

【3. 高血圧もコレステロールも「体の防衛反応」】
・心臓は約10秒に1回、全身の血管と連絡を取り合って「酸素が届いたか？」を確認している
・末端から「届いてません！」と返事が来ると、心臓は圧力を上げて無理やり届けようとする → これが高血圧の正体
（例）心臓が悪いのではなく、「末端に届かないから結果として圧を上げている」という体の防衛反応。むしろ体が一生懸命あなたを守ってくれている証拠
・高い圧力で血液を流し続けると血管が傷つく → そこに駆けつけて傷口をペタッと塞ぐのがコレステロール
（例）コレステロールは「悪玉」と呼ばれているが、実際の仕事は「血管の修復パテ（接着剤）」。数値が高い＝体のあちこちで血管が傷ついて修復工事中というサイン
・マッサージに行っても治らない慢性的な肩こりも、血圧を上げて血液を流すときに血管が破裂しないよう、周りの筋肉をギュッと固めてガードしている防衛反応。血流が改善すれば体が自然とガードを解いてくれる


【今日からやること】

・今日 → 「赤血球は変形して酸素を届けている」という話を、家族に1つ話してみる。人に話すと知識が自分のものになる
・3日以内 → クエン酸を意識して摂る。レモン水でも梅干しでもOK。クエン酸は赤血球の柔軟性を一時的に高めてくれることが確認されている
・1週間後 → 気になったら測定会で血流の状態を確認してみる。自分の赤血球が「ちゃんと変形して通れているか」を実際に目で見ることができる


〜ひとことメッセージ〜
数値は「敵」ではなく、体が自分を守ろうとしている証拠。仕組みがわかれば、やるべきことが見えてきます。
