【B07：有害金属とデトックスの仕組み】

〜知っておくと読みやすい用語〜
・有害金属 → 水銀・鉛・カドミウム・ヒ素・アルミニウムなど、体に悪影響を与える金属の総称
・水溶性（すいようせい）→ 水に溶ける性質。水溶性の毒は尿として比較的早く体外に出る
・脂溶性（しようせい）→ 油（脂肪）に溶ける性質。脂溶性の毒は脂肪に溜まり、体から出ていくのに何年もかかる
・デトックス → 体に溜まった老廃物や毒素を排出すること
・メタロチオネイン → 体の中で金属を運ぶタンパク質。本来は亜鉛を運ぶが、有害金属に乗っ取られることがある
・亜鉛（あえん）→ 細胞の修復や免疫に欠かせないミネラル。有害金属の排出にも深く関わる

〜この動画でわかること〜
「痩せない」「むくむ」の原因が、体に溜まった有害金属への"防衛反応"かもしれない。本当のデトックスの仕組みがわかります。


【今日の3つのポイント】

【1. 有害金属は脂肪に溶け込んで、数十年も体内に居座る】
・毒には大きく2つのタイプがある
（例）1つ目は「水溶性の毒（急性）」。フグ毒のように非常に強力だが、水に溶ける性質なので生き延びさえすればおしっこで比較的早く体の外に出ていく
（例）2つ目が問題の「脂溶性の毒（慢性）」。水銀・ヒ素・鉛・カドミウム・アルミニウムなどの有害金属。油（脂肪）に溶け込んで体の中にへばりつく。カドミウムが体から半分出ていくまでに15〜38年。子供の頃に入った金属が大人になっても残り続ける
・すぐには症状が出ないが、じわじわと体に影響を与え続ける

【2. 「痩せない」「むくむ」のは、毒を薄めようとする体の防衛反応】
・毒の影響は体重に反比例する → 体が大きいほど毒は薄まる
・許容量を超えると、体は命を守るために「体を大きくして薄めよう」とする
　①水分を溜め込んで毒の濃度を下げる → 慢性的な「むくみ」の正体
　②脂肪細胞を増やして毒を隔離する → 「なかなか取れない脂肪」の正体
（例）毒が溜まったまま無理にダイエットをすると、体からすれば「せっかく薄めて守っているのに、濃度を上げないで！」と抵抗する。これが食べていないのに痩せない・すぐリバウンドする原因の一つ
・出口さえ整えてあげれば、体は安心して脂肪を手放せるようになる。毒を出すルートを作ることが本当のダイエットの第一歩

【3. デトックスの本丸は汗ではなく「便」と「尿」】
・「岩盤浴で汗を流してデトックス！」とよく聞くが、汗から出る毒素は全体のわずか数パーセント
・メインの排出ルート：便が約75%、尿が約20%
・対策の3本柱：
　① 便通を良くする（最大の出口）
　② 亜鉛などの良いミネラルを摂る（有害金属を押し出す）
　③ 肝臓と腎臓をいたわる（解毒工場を元気にする）
（例）体の中には金属を運んでくれる「タクシー（メタロチオネイン）」が走っている。本来このタクシーには体に必要な「亜鉛」が乗るはずだが、カドミウムなどの有害金属は亜鉛より結合力が強く、タクシーを乗っ取ってしまう。良いミネラル（亜鉛）をたくさん入れて、タクシーを占拠している悪い金属を押し出すイメージ


【今日からやること】

・今日 → 食事に食物繊維を1品プラス。わかめの味噌汁、きんぴらごぼう、何でもOK。便通の改善がデトックスの第一歩
・3日以内 → 水分を意識して摂る。目安は1日1.5リットル以上。お水やお茶をこまめに飲んで尿からの排出をスムーズにする
・1週間後 → 亜鉛を含む食べ物を意識する。牡蠣・レバー・納豆・卵など身近な食材にたくさんある


〜ひとことメッセージ〜
出すルートさえ整えれば、体はちゃんと応えてくれます。本当のダイエットは「毒を出す道づくり」から始まります。
