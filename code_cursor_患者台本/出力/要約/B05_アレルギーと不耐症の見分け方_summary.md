【B05：アレルギーと不耐症の見分け方】

〜知っておくと読みやすい用語〜
・アレルギー → 免疫が「敵だ！」と誤認して攻撃する反応。花粉症や蕁麻疹（じんましん）などが代表的
・不耐症（ふたいしょう）→ 免疫とは無関係。食べ物を分解する酵素が足りず、うまく消化できない状態
・酵素（こうそ）→ 食べ物を細かく分解する「ハサミ」のようなもの。消化酵素は唾液や胃液にも含まれる
・アナフィラキシー → 重いアレルギー反応。全身に症状が出て、命に関わることもある
・浸透圧（しんとうあつ）→ 水分が濃度の高いほうへ移動する力。腸に未消化物があると水分がそこに集まる
・リーキーガット → 「腸漏れ」のこと（B04で詳しく解説）。腸の壁に穴が空いて毒素が体内に漏れる状態

〜この動画でわかること〜
「食べると調子が悪い」の正体は、アレルギーではなく「不耐症（消化酵素の不足）」かもしれない。その違いと対策がわかります。


【今日の3つのポイント】

【1. 大人の「アレルギーもどき」の多くは「不耐症」の可能性がある】
・アレルギー ＝ 免疫システムの暴走。体が「敵だ！」と認識して攻撃する反応。少量でも蕁麻疹やアナフィラキシーが起きることがある
・不耐症 ＝ 免疫はまったく関係ない。その食べ物を分解するための「ハサミ（酵素）」を持っていない、あるいは足りていない状態
（例）牛乳を飲むとお腹がゴロゴロする → これは「牛乳アレルギー」ではなく、乳糖を分解するハサミ「ラクターゼ」が足りない「乳糖不耐症」。日本人の約75%はこの酵素が不足している
（例）お酒が飲めない人も同じ仕組み。アルコールを分解する酵素がないから体がつらくなる ＝「アルコール不耐症」
・アレルギーなら「避ける」しかないが、不耐症なら「分解できる体に整えていく」ことで食べられるようになる可能性がある

【2. 分解できなかった食べ物は、36.5度のお腹の中で腐敗して毒素になる】
（例）体温は36.5度、真夏日と同じ気温。そこに消化できなかったお肉やパンを数時間放置したら…当然腐る。お腹の中で消化しきれなかった食べ物が腐敗して、ガスが発生し、毒素が出る。これがお腹の張りや原因不明の体調不良の正体
・腸は水を通すフィルター（半透膜）のような構造。未消化物がたまると浸透圧の原理で体中の水分がそこに引っ張られる
（例）ナメクジに塩をかけると水分が抜けてしぼむ。あれと同じ原理で、腸に消化できないものがあると体中の水分が腸に集まって「下痢」が起きる
・腸内環境が悪化 → リーキーガット（腸漏れ） → 毒素が全身をめぐり → 皮膚から出ようとして湿疹やアトピーの悪化に

【3. 対策は「何を食べるか」ではなく「消化能力以上に食べないこと」】
・一番の対策は「食べる量を減らすこと」。酵素の量を超えて食べているから処理しきれずに腐る。キャパシティオーバーが原因なら入れる量を減らすのが鉄則
・「よく噛むこと」＝ 無料で使える最強の消化サポート。唾液には消化酵素が含まれていて、よく噛むだけで胃腸に届く前にかなりの消化が進む
・余裕があれば生野菜や発酵食品で酵素を補う
・「何を食べたら治りますか？」とよく聞かれるが、正解は足し算の前に引き算のケア。「いかに消化できる量に抑えるか」が先


【今日からやること】

・今日 → 次の食事で「いつもよりちょっと少なめ」を意識する。腹八分目でストップ。それだけでお腹の中の腐敗リスクはぐっと下がる
・3日以内 → 「一口ごとに20〜30回噛む」を試してみる。慣れると自然になる。噛むだけで酵素がたっぷり出てくれる
・1週間後 → お腹の張りや肌の調子がどう変わったか振り返ってみる。「なんかいつもと違うかも」と感じたら体がちゃんと応えてくれている証拠


〜ひとことメッセージ〜
「体質だから」と諦めなくていい。足し算の前に引き算のケア。食べる量を減らしてよく噛むだけで、体は変わり始めます。
