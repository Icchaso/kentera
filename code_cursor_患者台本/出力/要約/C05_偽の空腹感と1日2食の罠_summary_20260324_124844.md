# C05：偽の空腹感と1日2食の罠

## 3行サマリー
- ゴール：間食がやめられない本当の原因（偽の空腹感）を理解し、「意志の弱さ」ではなく「血糖値の仕組み」の問題だとわかる
- 視聴後にできること：午後の間食衝動を「偽物の食欲」だと見抜けるようになり、3食を基本にした食事リズムを始められる
- CTA：明日から3食食べる。朝食を抜いている方はおにぎり1個から始める

## 主要ポイント
1. 間食がやめられないのは「意志が弱い」からではなく、血糖値の急降下が脳を騙しているだけ
2. 朝食抜きの「1日2食」は、お相撲さんが太るためのメソッドと同じ — ダイエットには逆効果
3. 3食しっかり食べて血糖値を穏やかに保つことが、無駄な食欲を消す一番の近道

## 各スライドの要点
- スライド1：間食の悩みへの共感 → パラダイムシフト「意志ではなく脳が血糖値に騙されている」→ 仕組みと解決法を伝える約束
- スライド2：結論の3つ（偽の空腹感の正体／1日2食の罠／3食で解決）を先出し
- スライド3：血糖値スパイクの急降下時に、体にカロリーが足りているのに脳が「エネルギー不足」と誤認する＝偽の空腹感の正体。ジェットコースターメタファーを本格展開
- スライド4：偽の空腹感→間食→再びジェットコースター→また偽の空腹感、という悪循環ループ。伏線回収「だから間食がやめられなかった」
- スライド5：食事間隔が空くほどジェットコースターが激化する。お相撲さんは太るために「1日2食」を採用。「食事回数を減らす＝逆効果」の伏線回収
- スライド6：3食に分けて食べることでジェットコースターが「凪」になり、偽の空腹感が消える
- スライド7：午後の空腹を感じたら「本物か偽物か」を問いかけ、まずコップ1杯の水を飲む
- スライド8：我慢ではなく3食食べることが本旨。チェックリスト（今日：水を飲む／3日：朝食を食べる／1週間：昼食の糖質偏りを見直す）
- スライド9：3つの結論をおさらい。メタファー最終コールバック「ジェットコースターから凪へ」。処方型CTA「明日から3食食べてみてくださいね」＋概要欄への軽い案内（Tier 2：内容と測定の結びつきが薄いため、さらっと一言）

## 今日からのアクション
- 今日：午後に「小腹が空いた」と思ったら、まずコップ1杯の水を飲む
- 3日以内：朝食を抜いている人は、おにぎり1個でいいので朝食を食べる
- 1週間後：昼食の内容を振り返り、糖質偏りがあればタンパク質・野菜をプラスする
