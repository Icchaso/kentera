こんにちは！今回のテーマは「リバウンドを終わらせる仕組み」です。
毎日ランニングを頑張っているのに痩せない、サラダ中心の食事にしても変わらない、話題のダイエット薬に手を出そうかと迷っている…。そんな経験、ありませんか？
実は、私たちの体には「自動調整システム」が備わっています。無理に痩せようとすると、体がちゃんと元に戻してしまうんです。太っているのは意志が弱いからではなく、体が毒から身を守るために脂肪をたくわえている"ぼうえいはんのう"かもしれません。
今日はその仕組みと、リバウンドをこんぽんから終わらせる方法をお伝えします。見終わるころには、ダイエットへの向き合い方がガラッと変わっているはずですよ。

結論からお伝えしますね。今日覚えることは3つです。
1つ目、運動でしょうひカロリーを増やしても、体がサーモスタットのように自動で帳消しにする。
2つ目、ししつカット薬はべんもれリスクがあり、やめればリバウンドする不自然な方法。
3つ目、リバウンドを終わらせるカギは、「体重を減らす」から「体を健康にする」へ目的を切り替えること。
この3つの裏にある「体の自動調整システム」がわかると、もう無理なダイエットに振り回されなくなりますよ。

それでは、さっそく本題にいってみましょう！
1つ目の「運動でしょうひカロリーを増やしても、体が自動で帳消しにする」というお話です。
まず、皆さんが気にしている「カロリー」について、意外な事実をお伝えしますね。
カロリーというのは、もともと「水の温度を上げるためのエネルギー」を表す単位なんです。食べ物に火をつけて燃やしたときの熱量をはかっているだけ。
当然ですが、私たちの体の中で食べ物に火がついてゴーゴーと燃えているわけではないんですよ。
私たちの体は、ただ食べたものを燃やすだけの「単純なしょうきゃくろ」ではありません。実はもっと賢い仕組みになっているんです。

最近の研究では、「運動だけで体重やたいしぼうを大きく減らすのは難しい」という考え方が主流になっています。
「えっ、走ればカロリーをしょうひして痩せるでしょ？」と思いますよね。
ここで、エアコンの「サーモスタット」を想像してみてください。部屋の温度が下がりすぎると、自動で暖房が強くなって設定温度に戻しますよね。
体もまったく同じなんです。激しい運動で一時的にしょうひカロリーが上がっても、体は「エネルギーを使いすぎた！」と判断して、脳の活動やほかの部分のしょうひをこっそり抑え、1日のトータルを元に戻そうとします。
体のサーモスタットが、増えたしょうひをほかの場所で帳消しにしていた。これが「自動調整システム」の1つ目です。だから、計算通りに一生懸命ランニングしても痩せなかったんですね。

さらに驚くデータがあります。同じ10キロメートルという距離を「走る」場合と「歩く」場合。トータルのしょうひカロリーはほとんど変わらないという実験結果が出ているんです。
カロリーをしょうひするためだけに歯を食いしばって走り続けるのは、体にも心にも負担が大きい割に、結果がついてきません。
でも安心してください。運動が「意味がない」のではありません。カロリーしょうひではなく、別の"目的"に変えてあげれば最高の味方になるんです。これは後ほど詳しくお伝えしますね。

さて、運動でカロリーをしょうひしてもサーモスタットで帳消しにされるなら、「じゃあ薬に頼ろう」と考えるかたがいます。その気持ちはよくわかります。頑張っても結果が出ないと、手っ取り早い方法がほしくなりますよね。でも、2つ目のポイントとして、これだけは知っておいてください。
最近、ししつのきゅうしゅうをカットする市販薬が話題になっていますが、医学的な観点から、これはおすすめできません。理由は、深刻なふくさようがあるからです。
この薬は、腸でししつのきゅうしゅうを邪魔します。ところが、私たちのこうもんのあたりには「油をかんちするセンサー」がありません。
きゅうしゅうされなかった油がそのまま腸を素通りしてくるとどうなるか。自分ではべんいを感じないまま、油が漏れ出してしまうんです。実際、この薬を使うときは「おむつやパッドをつけてください」と指導されます。
おむつを履いてまで、一時的に薬で痩せたいですか？ダメですよ〜、不自然な方法に頼っちゃ。
これもサーモスタットと同じなんです。薬で無理やりししつをカットしても、体の自動調整システムそのものは変わっていない。だから薬をやめれば、当然リバウンドするんです。

さあ、ここからが一番大切なお話ですよ。3つ目の「体重を減らす」から「体を健康にする」へ目的を切り替える、というお話です。
そもそも、なぜ私たちは太ってしまうのか。ここに体のすごい秘密が隠されているんです。
以前の動画でゆうがいきんぞく、つまりまんせいてきな毒のお話をしたのを覚えていますか？
体に毒が溜まると、毒の影響は体重にはんぴれいします。つまり体は命を守るために「毒を薄めよう」として、水分を溜め込んでむくんだり、脂肪をたくわえて毒をかくりしようとしたりするんです。
ここでてんびんをイメージしてみてください。片方のお皿に毒が乗っかると、バランスを取るために反対側のお皿に水分や脂肪を乗せて釣り合わせようとする。これが体の自動調整システムの2つ目、「てんびんのバランス」です。
そしてここが最大のポイントです。てんびんの片方にまだ毒が乗ったまま、反対側の脂肪だけを無理やり減らそうとするとどうなるか。体は「危ない！バランスが崩れる！」と感じて、必ず脂肪を元に戻そうとします。
だからリバウンドしていたんです。これがリバウンドの正体なんですね。
ちょっとドキッとしますよね。でも、知っているだけで未来は変えられます。

やるべきことはシンプルです。
まず、「体重を減らす」という目的を「体を健康にする」に切り替えること。
次に、カロリー計算や薬に頼る習慣から、少しずつ離れていくこと。長年カロリーを気にしてきたかたにとって、いきなり手放すのは不安ですよね。でも、サーモスタットが帳消しにしてしまう以上、数字を追い続けても遠回りになってしまうんです。
そして、食後1時間以内に軽く体を動かして、けっとうちの急上昇を穏やかにすること。
さらに、ミネラルをしっかり摂って、てんびんの片方に乗っている毒を安全に体の外に出してあげることが大切なんです。
覚えていますか？ポイント1で「運動の目的を変えれば最高の味方になる」とお伝えしましたよね。ここで繋がるんです。カロリーを燃やすために走るのではなく、けっとうちの急上昇を抑えるために食後に歩く。同じ「運動」でも、目的が変わればまったく別の意味を持つんですよ。

ここまでの話を聞いて「自分の体は大丈夫かな…」と感じた方もいるかもしれません。不安になる必要はありません。体はちゃんと応えてくれる力を持っていますよ。
もちろん、持病のある方や医師の指導を受けている方は、お医者さまの方針を優先してくださいね。今日の内容はアクマで、日々の生活で取り入れられる「判断軸」です。気になることがあれば、いつでもお医者さまや私たちに気軽に相談してくださいね。
では、今日から何をすればいいか。具体的なアクションをお伝えしますね。
まず今日やること。次の食事のあと、10分だけ散歩してください。カロリーを燃やすためではなく、けっとうちの急上昇を穏やかにするためです。目的を変えるだけで、同じ「歩く」がまったく別の意味を持ちます。
次に3日以内。カロリー計算を気にする代わりに、「しょくもつせんい」と「ミネラル」が入った食事を意識してみましょう。いきなり計算をゼロにしなくても大丈夫です。まずは「数字よりも中身」に目を向けるところから始めてみてください。
まずはこの2つから、焦らず始めてみてくださいね。できない日があっても、翌日からまた意識すればそれで十分ですよ。

最後に、今日の3つをおさらいしましょう。
1つ目、運動でしょうひカロリーを増やしても、体がサーモスタットのように自動で帳消しにする。
2つ目、ししつカット薬はべんもれリスクがあり、やめればリバウンドする不自然な方法。
3つ目、リバウンドを終わらせるカギは、「体重を減らす」から「体を健康にする」へ目的を切り替えること。てんびんの片方に乗っている毒を取り除けば、体は自然と本来のバランスに戻っていきます。
サーモスタットに逆らわず、てんびんのバランスを整える。それがリバウンドを終わらせる唯一の道です。

そして最後に1つだけ。今日お伝えした自動調整システムの仕組みがわかると、「じゃあ私のてんびんには何が乗っているの？」と気になりますよね。実は、そこが一番大事なところなんです。
同じリバウンドでも、てんびんの片方に乗っている毒の種類も量も、ひとりひとりまったく違います。今日のテーマは、まさに「測定」で明らかになるこんぽんげんいんと直結していますので、測定での答え合わせはぜひ強くおすすめしたいです。
体が何から守ろうとしているのかがわかれば、正しい対策が見えてきます。せっかく仕組みをりかいできたのに、自分の体に合わないことをやっていたらもったいないですからね。
あなたの体は、ちゃんと応えてくれる力を持っています。その力を正しく引き出す最短ルートを一緒に見つけていきましょうね。
