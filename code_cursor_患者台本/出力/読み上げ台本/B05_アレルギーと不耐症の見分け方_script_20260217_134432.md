スライド6: ポイント① アレルギーと不耐症の決定的な違い
ナレーション

まず「アレルギー」。これは免疫システムの暴走です。
前回お話ししたように、体が「敵だ！」と認識して攻撃をしかける反応ですね。
少量でも反応が出ることがあり、蕁麻疹（じんましん）が出たり、ひどい場合はアナフィラキシーが起きたりします。[ポーズ]
一方で「不耐症」。こちらは免疫はまったく関係ありません。
単純に、その食べ物を分解するための「酵素」…いわば「ハサミ」を持っていない、あるいは足りていない状態なんです。[ポーズ]
ここ、知ると世界の見え方が変わるポイントですよ。

スライド7: ポイント① 乳糖不耐症という身近な例
ナレーション
一番わかりやすい例が「牛乳」です。[ポーズ]
牛乳に含まれる「乳糖（にゅうとう）」という成分を分解するハサミ、「ラクターゼ」という酵素を、日本人の約75%は十分に持っていないと言われています。[ポーズ]
だから、牛乳を飲むとお腹を下す。
これは「牛乳アレルギー」ではなくて、単に分解できずに通過しているだけの「乳糖不耐症」なんですね。[ポーズ]
お酒が飲めない人がいるのと同じ仕組みです。アルコールを分解する酵素がないから体がつらくなる。これも「アルコール不耐症」ですよね。[ポーズ]
ここからが大事なところですよ。
アレルギーなら、その食べ物を「避ける」しかありません。
でも不耐症なら、「分解できる体に整えていく」ことで、食べられるようになる可能性があるんです。[ポーズ]
でもね、これって逆に言えばチャンスでもあるんですよ。
「体質だから仕方ない」と諦める必要はないかもしれない、ということですからね。

スライド8: ポイント② 36.5度の真夏日と腐敗
ナレーション

では、酵素が足りなくて分解できなかった食べ物が、お腹の中に残るとどうなるでしょうか？
ここでちょっと強烈なイメージをしてみてください。[ポーズ]
私たちの体温は36.5度くらい。真夏日と同じ気温ですよね。
そこに、消化できなかったお肉やパンなどを数時間放置したら、どうなると思いますか？[ポーズ]
当然…腐りますよね。[ポーズ]
お腹の中で、消化しきれなかった食べ物が腐敗して、ガスが発生し、毒素が出る。
これが、お腹の張りや不快感、そして原因不明の体調不良の正体なんです。

スライド9: ポイント② ナメクジに塩の原理
ナレーション
さらに面白い仕組みがあるんです。[ポーズ]
腸は「半透膜（はんとうまく）」といって、水を通すフィルターのような構造をしています。
分解できなかった大きなゴミ、つまり未消化物が腸にたまると、「浸透圧（しんとうあつ）」の原理で、体中の水分がその場所に引っ張られるんですよ。[ポーズ]
イメージしやすい例を一つ。ナメクジに塩をかけると、水分が抜けてしぼみますよね。あれと同じ原理です。
腸に消化できないものがあると、体中の水分が腸に集まってきて、その結果「下痢」が起きるんです。[ポーズ]
つまり、肌荒れやアトピーが悪化しているとき、あなたのお腹の中では「真夏日に生ゴミを放置しているような状態」が起きているかもしれないということなんですね。[ポーズ]
聞くとびっくりしますよね。でも、だからこそ体には伸びしろがあるんです。この「腐敗」を止めれば、体は変わっていけるということですからね。

スライド10: ポイント③ リーキーガットと皮膚の炎症
ナレーション
ではもう少しだけ深掘りしてみましょう。3つ目のポイントです。[ポーズ]
こうして腸内環境が悪化して、未消化の食べ物や毒素が増えると、腸の壁がダメージを受けて穴が空いてしまいます。
前回の動画でお話しした「リーキーガット症候群（腸漏れ）」ですね。[ポーズ]
本来なら便として出るはずの毒素や、未消化のタンパク質が、穴から血管に侵入して全身をめぐります。
そして、皮膚という「出口」から出ようとした結果が、湿疹やアトピー性皮膚炎の悪化として現れるんです。[ポーズ]
ここまで聞いて「えっ」と思った方、その気づきがもう第一歩なんですよ。

スライド11: ポイント③ 対策は「引き算」
ナレーション
「じゃあどうすればいいの？」ということですよね。[ポーズ]
薬を塗ったり、特定の食べ物を避けたりするのも大切です。でもまずは、体の中の「腐敗」を止めることが先決なんです。[ポーズ]
一番の対策は、とてもシンプル。
まず、「食べる量を減らすこと」。
自分の消化能力、つまり酵素の量を超えて食べているから、処理しきれずに腐ってしまう。キャパシティオーバーが原因なら、入れる量を減らすのが鉄則ですよね。[ポーズ]
そして、「よく噛むこと」。
唾液には消化酵素が含まれています。よく噛むだけで、胃腸に届く前にかなりの消化が進むんですよ。
噛むことは、無料で使える最強の消化サポートなんです。[ポーズ]
さらに余裕があれば、生野菜や発酵食品で酵素を補ってあげると、なお良いですね。[ポーズ]
「何を食べたら治りますか？」とよく聞かれるんですが、正解は「何を食べるか」の前に「いかに消化できる量に抑えるか」。
足し算の前に、引き算のケアを意識してみてくださいね。
