こんにちは！
今回のテーマは「血糖値」です。健康診断で「血糖値がちょっと高めですね」と言われて、モヤモヤしているかた、いらっしゃいませんか？
あるいは、お昼ごはんのあと、気絶するように眠くなって午後がつらい…。
「意志が弱いからかな」「食べすぎたかな」って思っていませんか？少し驚くかもしれませんが、実はそこに希望のタネが隠れているんです。
今日はその仕組みを、わかりやすくお話ししますね。

今日は、血糖値が乱れる"なぜ？"をスッキリ解消していきます。しかも、一般的に言われる「インスリン」の話だけじゃなくて、ちょっと意外な「もう一つの犯人」についてもお伝えしますよ。
これ、聞くと「え、そうなの！？」ってなるかたがとても多いんです。この動画が、皆さんの「はじめの一歩」を後押しできたら嬉しいです。

結論を先にお伝えしますね。今日覚えていただきたいのは、たった3つです。1つ目、食後の眠気は「血糖値スパイク」のサイン。血管が傷ついている可能性がある。
2つ目、血糖値が下がらないのは、下げる力の問題だけじゃない。上げるホルモンの暴走も関係している。
3つ目、運動の目的は「カロリー消費」ではなく、「食後の血糖値を抑えること」。この3つがわかると、明日からの行動がハッキリ変わりますよ。
では、順番に見ていきましょうね。

ところで皆さん、「糖質制限がいい」「いや、やっぱりバランスよく」「運動しなきゃ」「サプリで解決」…。
血糖値に関する情報って、本当にたくさんありますよね。正直なところ、何が正解なのかわからなくなりませんか？でも、不安になる必要はありませんよ。
体の中で起きている「仕組み」さえわかれば、自分にとっての優先順位が自然と見えてくるんです。

前回の動画で、栄養の「土台」を整えることが大事だとお話ししましたよね。今日のテーマである血糖値も同じです。
「あれをやめろ」「これを食べろ」という対策の前に、まず「なぜ血糖値が乱れるのか」という仕組みを押さえましょう。仕組みがわかれば、「じゃあ自分はここから始めよう」が自然と決まります。
お待たせしました。ここからがほんぺんですよ。

はい、1つ目の「食後の眠気は血糖値スパイクのサイン」というお話です。私たちが食事、特にごはんやパンなどの炭水化物を食べると、血液の中の糖分、つまり血糖値が上がります。
これ自体は、体にエネルギーを届けているだけなので問題ありません。
ところが、精製された糖質…白いごはん、菓子パン、甘いジュースなどを一気に食べると、血糖値が「ドーン！」と急上昇するんです。
すると体は「大変だ、血管が砂糖漬けになっちゃう！」と慌てて、血糖値を下げるホルモン「インスリン」を大量に出します。

大量のインスリンによって、今度は血糖値が「ドーン！」と急降下する。この急上昇からの急降下。まるでジェットコースターですよね。
これを「血糖値スパイク」と呼びます。

血糖値が急降下すると、体は低血糖に近い状態になります。
これが、食後のあの強烈な眠気やだるさ、イライラの正体なんです。つまり、食後の眠気は「やる気の問題」でも「性格の問題」でもないんですね。
体の中で血糖値のジェットコースターが起きているサインなんです。そしてもう一つ、大事なことがあります。
第一回の動画で「血管の内側の壁が命」というお話をしたのを覚えていますか？この血糖値の急激な上下動は、血管の内側をガリガリと傷つけてしまうんです。
動脈硬化の大きな原因の一つでもあります。でもね、この仕組みがわかったかたはもう一歩リードしていますよ。
スパイクを起こさないようにすればいい。その方法は、ちゃんとあるんです。

さて、ここからがさらに面白くなりますよ。2つ目の「上げるホルモンの暴走」についてです。糖尿病というと、「インスリンが出なくなる病気」「インスリンが効かなくなる病気」と思いますよね。
もちろん、それも正解です。でも実は最近の研究で、もう一つの視点が注目されています。血糖値を下げるのがインスリンなら、血糖値を上げるホルモンもあるんです。
その名前は「グルカゴン」。
本来、食事をして血糖値が上がったら、体は「もう糖分はいらないよ」と判断して、このグルカゴンの分泌を止めます。ところが…ちょっと不思議な話なんですが、聞いてみてください。

糖尿病やその予備軍のかたの一部では、食事をして血糖値が上がっているのに、なぜかグルカゴン、つまり「上げるほうのホルモン」が出続けてしまっている。
そんなことがわかってきたんです。イメージとしては、車のアクセルとブレーキを同時に踏んでいるような状態です。
ブレーキ役のインスリンがいくら頑張っても、アクセル役のグルカゴンが踏み込まれていたら、血糖値は下がりませんよね。体はどんどん疲弊してしまいます。でもここで諦める必要はありません。
この乱れたバランスを整える方法が、実はとても身近なところにあるんです。
次のポイント、これもぜひ知っておいてほしい話ですよ。

3つ目は「運動の目的はカロリー消費ではなく、血糖値を抑えること」というお話です。皆さん、運動と聞くと「ダイエットのためにカロリーを消費しよう」と思いませんか？
でも実は、運動で消費できるカロリーって、びっくりするほど少ないんですよ。
おにぎり1個分のカロリーを消費するのに、何十分も走らなきゃいけない。なので今日から、考え方をガラッと変えてみましょう。
運動の目的は、カロリー消費ではありません。
「血糖値を物理的に下げること」なんです。筋肉を動かすと、インスリンの助けを借りなくても、筋肉が直接血液中の糖を取り込んでくれるんですよ。
これを知ったとき、私もすごくワクワクしたんですよね。

そして大事なのが、運動をするタイミングです。おすすめは「食後1時間」。
ちょうど血糖値がピークを迎えようとするこのタイミングで、散歩やスクワットをするんです。そうすると、上がりすぎようとする血糖値の山を、筋肉の力で物理的に低く抑えることができます。
山が低くなれば、インスリンを大量に出す必要もなくなる。
急降下も起きないので、食後の眠気も防げるんです。「食べたら1時間後に動く」。
シンプルですけど、これが血糖コントロールの鉄則なんですよ。

ここまでのお話を聞いて、「自分、糖尿病なのかな…」と心配になったかたがいるかもしれません。もちろん、お医者さまの検査や治療はとても大切です。それを前提にお話ししていますからね。

この動画は、診断でも治療のアドバイスでもなく、あくまで「体の仕組みを知るための判断軸」をお渡ししているものです。
血糖値の数値が気になるかたや、食後の強い眠気がずっと続くかたは、ぜひ一度お医者さまに相談してみてくださいね。そのうえで、今日お話しした「食後の運動」は、どなたでも安心して始められる習慣です。
完璧を目指さなくていいんです。「ちょっと意識してみようかな」、それだけでじょうできですよ。

それでは、今日から何をすればいいか。チェックリストをお伝えしますね。まず今日やること。次の食事のあと、「食後1時間」を意識して、5分でいいので歩いてみてください。階段の上り下りだけでもOKです。難しく考えなくて大丈夫、それだけで立派な第一歩ですよ。

次に3日以内。食事の順番を少し変えてみましょう。ごはんやパンの前に、野菜やおかずを先に食べる。これだけで血糖値の急上昇がゆるやかになることがわかっています。

そして1週間後。食後の眠気がどう変わったか、ぜひ振り返ってみてください。「あれ、前より楽かも」と感じたら、体がちゃんと応えてくれている証拠です。

今日の話を聞いて、「もっと自分の体のことを知りたいな」と思ったかた。ここまでで、「血液」「栄養」「血糖値」というきそがためのSTEP1が終わりました。
次回からは【STEP2】に入ります。テーマは「免疫系」です。「免疫力アップ」とよく言いますが、実はアレルギーなどは免疫が暴走している状態なんですよ。
なぜ免疫が暴走するのか？その鍵を握る「腸のバリア」についてお話ししていきますので、ぜひ次回の動画もチェックしてみてくださいね。

また、「自分の血糖値の状態が気になるな」というかたは、測定会で実際の状態を確認することもできますので、気になるかたは覗いてみてください。

最後に、今日の内容を3つだけおさらいしましょう。1つ目、食後の強い眠気は「血糖値スパイク」のサイン。血管が傷ついている可能性があるから、見逃さないでいきましょう。
2つ目、血糖値が高いのは、インスリンの問題だけでなく、グルカゴンという「上げるホルモン」の暴走も関係している。
3つ目、運動はカロリー消費ではなく、「食後1時間に血糖値を抑える」ために行う。次にやることは1つだけ。今日の食事のあと、食後1時間にちょっと歩いてみてくださいね。
今日のうちにそれをやってみると、明日の自分がちょっと変わりますよ。焦らず、一つずつ。体はちゃんと応えてくれますからね。
