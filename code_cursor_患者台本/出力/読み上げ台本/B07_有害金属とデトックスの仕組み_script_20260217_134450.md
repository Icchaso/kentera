結論を先にお伝えしますね。今日覚えていただきたいのは3つです。1つ目、有害金属は脂肪に溶け込んで、数十年も体内にいすわる。
2つ目、「痩せない」「むくむ」のは、毒を薄めようとする体の防衛反応かもしれない。
3つ目、デトックスの本丸は汗ではなく「便」と「尿」。そして亜鉛などの必須ミネラル。今日の話がわかると、ダイエットやデトックスに対する考え方がガラッと変わりますよ。
それでは、さっそく本題にいってみましょう。

ところで皆さん、「デトックス」で調べると、ものすごい量の情報が出てきますよね。「岩盤浴で汗を流してデトックス」「デトックスウォーターを飲もう」「ファスティングで毒を出そう」…。
どれが本当なのか、何から始めればいいのか、わからなくなっていませんか？でも安心してくださいね。体の仕組みがわかれば、正しい優先順位が自然と見えてきますから。

大事なのは、「あれがいい」「これがいい」という方法論の前に、まず「毒とは何で、体の中でどう振る舞うのか」を知ることなんです。仕組みさえわかれば、「じゃあ自分はここを整えよう」が決まります。
正解探しをやめて、優先順位で勝つ。今日はその判断軸をお渡ししますね。いいですか、ここからが大事なところですよ。

1つ目の「有害金属は脂肪に溶け込んで数十年いすわる」というお話です。まず、「毒」には大きく2つのタイプがあることを知ってください。1つ目は「水溶性の毒」。これは急性タイプです。
例えば、フグ毒。非常に強力で、体に入ると短時間で命に関わります。
でも水に溶ける性質なので、生き延びさえすればおしっことして比較的早く体の外に出ていくんですね。問題は2つ目、「脂溶性の毒」。こちらは慢性タイプです。
これが今日のテーマである、水銀・ヒ素・鉛・カドミウム・アルミニウムなどの「有害金属」です。
ちなみに、昔は「重金属」と呼ばれていましたが、アルミニウムのように軽い金属も含むので、今は「有害金属」と呼ぶのが正確なんですよ。

ここ、知ると見え方が変わるポイントなんです。有害金属は、摂取してもすぐには症状が出ません。
その代わり、油…つまり脂肪に溶け込んで、体の中にへばりつくんですね。どれくらい長くかというと、例えばカドミウムが体から半分出ていくまでに、なんと15年から38年もかかるんです。
子供の頃に入った金属が、大人になっても、おじいちゃん・おばあちゃんになっても残り続けている。ちょっとドキッとしますよね。
でも、この仕組みがわかったということは、対策の方向が見えたということでもあるんですよ。

では、2つ目の「痩せない・むくむのは体の防衛反応」についてです。ここが今日の核心部分です。なぜ有害金属が溜まると太るのか。
ここで「毒の濃度と体重の関係」をイメージしてみてください。毒の影響は、体重に反比例します。
つまり、体が大きければ大きいほど、毒は薄まりますよね。例えば、あなたの体に許容量ギリギリの毒が溜まっているとします。
そこに、さらに食事から金属や添加物が入ってきたらどうなるか。
許容量を超えてしまって、体が危険にさらされます。

この時、体は命を守るために究極の選択をするんです。「毒を出せないなら、体を大きくして薄めよう」。
まず、水分を溜め込んで毒の濃度を下げます。これが慢性的な「むくみ」の正体なんですね。
次に、脂溶性の毒を閉じ込めるために脂肪細胞を増やして、そこに毒を隔離します。これが「なぜか取れない脂肪」の正体です。
つまり、脂肪やむくみは、ただの「太っている」ではなくて、体が毒から内臓を守ろうとしている防衛反応かもしれないんです。
だから、毒が溜まったまま無理にダイエットをすると、体からすれば「せっかく薄めて守っているのに、濃度を上げないで！」と抵抗します。
これが、食べていないのに痩せない・すぐリバウンドする原因の一つなんですよ。でもね、出口さえ整えてあげれば、体は安心して脂肪を手放せるようになるんです。
毒を出すルートを作ってあげることが、本当のダイエットの第一歩なんですよ。

さぁ、3つ目は「デトックスの本丸は便と尿」というお話です。では、どうすればこの毒を出せるのか。「岩盤浴で汗をかいてデトックス！」とよく聞きますが、実は汗から出る毒素は全体のわずか数パーセントなんです。
メインの出口は「便」が約75%、「尿」が約20%。意外ですよね。つまり、デトックスの本丸は「しっかり出す」こと。
汗をかくことが悪いわけではありませんが、まず便通と排尿をきちんと整えることのほうが、ずっと大事なんです。

もう一つ、大事な話がありますよ。体の中には、金属を運んでくれるタクシーのようなタンパク質が走っています。
名前は「メタロチオネイン」。難しい名前ですが、覚えなくて大丈夫ですよ。タクシーだと思ってください。本来、このタクシーには体に必要な「亜鉛（あえん）」が乗るはずなんです。
ところが、カドミウムなどの有害金属は亜鉛よりも結合力が強いので、タクシーを乗っ取ってしまうんですね。
その結果、亜鉛が行き届かなくなって、味覚がおかしくなったり、肌が荒れたり、気分が落ち込みやすくなったりします。
じゃあどうするか。対策は3つです。
1つ目、便通を良くすること。最大の出口は「便」ですからね。
2つ目、亜鉛などの良いミネラルをしっかり摂ること。良いミネラルをたくさん入れて、タクシーをせんきょしている悪い金属を押し出すイメージです。
3つ目、解毒工場である肝臓と腎臓をいたわること。この3つが整って初めて、脂肪の中に隠れていた毒が外に出ていき、「痩せやすい体」へと変わっていくんですね。

ここまでのお話を聞いて、「自分の体にも毒がいっぱい溜まっているのかな…」と心配になったかたもいるかもしれません。焦る必要はまったくありませんよ。
有害金属は、現代に生きている以上、誰の体にも多少は入っています。
