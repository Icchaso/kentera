こんにちは！
今回のテーマは「コレステロールと血圧」です。健康診断の結果を見て、「コレステロールが高い」「血圧が上がってきた」と言われて、なんだかモヤモヤしているかた、いらっしゃいませんか？
減塩してみたり、油ものを控えてみたり…自分なりに頑張っているのに、なかなか数値が良くならない。そんな経験はありませんか？でもここからが面白いところで、実は解決の糸口もすぐそばにあるんですよ。

今日の内容がわかると、コレステロールや血圧への向き合い方がガラッと変わります。「悪玉コレステロールを下げなきゃ」「とにかく減塩しなきゃ」…そんな思い込みが、ちょっとひっくり返るかもしれません。
一緒にその仕組みを覗いてみましょう。きっと面白い発見がありますよ。

結論を先にお伝えしますね。今日覚えていただきたいのは3つです。1つ目、悪玉コレステロールは「血管の修復パテ」。数値が高いのは、血管が傷ついている証拠。
2つ目、高血圧は、末端まで血液を届けるための防衛反応。むやみに下げると末端が酸欠になってしまう。
3つ目、対策は減塩よりも、野菜でカリウムをしっかり摂ること。見終わるころには、「数値は下げるもの」から「読みとくもの」へ、考え方がきっと変わっていますよ。

ところで皆さん、コレステロールや血圧の数値が高いと言われたとき、まず何を考えますか？「薬を飲まなきゃいけないのかな」「食事をもっと制限しなきゃ」…多くのかたがそう思うんですよね。
テレビでも「悪玉コレステロールを下げよう」「減塩で血圧改善」と繰り返し言っていますから、そうなるのも無理はありません。でも安心してくださいね。ここにはちゃんと改善のヒントがあるんです。

実は、数値が高いことそのものが「悪」なのではなくて、体は理由があってあえて数値を上げているんです。例えるなら、火事の現場に駆けつけた消防車。
「消防車がたくさんいるから、あの場所は危ない」と思いますよね。でも消防車が悪者ではないですよね。
火を消しに来てくれているんです。コレステロールも血圧も同じ。体を助けに来ている「味方」なんです。
ここ、聞くと数値の受け止め方がガラッと変わるポイントですよ。それでは、さっそく本題にいってみましょう！

1つ目の「悪玉コレステロールは血管の修復パテ」というお話です。LDLコレステロール、いわゆる「悪玉」。実はこの「悪玉」という呼び名、医学用語ではないんです。メディアがわかりやすくするためにつけた名前なんですね。
では、LDLコレステロールの本当の役割は何なのか。実は彼ら、「血管の修復パテ」、つまり補修材なんです。
私たちの体って、本当によくできてるんですよね。

血管の内側は、高血糖やストレス、慢性的な炎症によって、日々少しずつ傷ついています。もし血管の壁が破れてしまったら、大変なことになりますよね。
そこで体は「あそこの壁が傷ついているぞ、急いで塞いでくれ！」と指令を出します。
駆けつけて傷口をペタッと塞いでくれるのが、LDLコレステロールなんです。つまり、コレステロール値が高いということは、「体のあちこちで血管が傷ついていて、今まさに修復工事中ですよ」というサインなんですね。

ここで大事なことをお伝えしますね。もし薬でコレステロール、つまり修復の材料だけを減らしてしまったらどうなるでしょう。
工事は中断されて、血管は傷ついたまま放置されてしまいますよね。下げるべきは数値ではなく、血管を傷つけている「炎症の原因」のほうなんです。でもね、ここに体の本当の賢さがあるんです。
炎症の原因に目を向けて、食事や生活を整えていけば、体は自然と修復を終えてコレステロールを減らしていきます。
原因にアプローチすれば、数値はあとからついてくるということなんです。

さあ、次のポイントにいきますよ。2つ目の「高血圧は防衛反応」についてです。私たちの体には約100億本以上の血管がありますが、心臓というポンプはたった1つしかありません。
実はこの心臓、10秒に1回、全身の細胞と連絡を取り合っているんです。
「おーい、酸素届いたかー？」って確認しているんですよ。ここに体のすごい秘密が隠されているんです。

もし血管が動脈硬化で硬くなっていたり、ドロドロの血液で詰まりかけていたら、末端の細胞から「酸素が届いていません！」という返事が来ます。すると脳はこう判断するんです。
「このままでは末端が死んでしまう。圧力を上げて、なんとか届けろ！」と命令を出します。
これが高血圧の正体なんですね。つまり高血圧も、末端の細胞を生かすための防衛反応なんです。
実はここ、知るとちょっと感動する話なんですよ。体って、自分を守るために必死に頑張っているんです。

はい、ここが今日の肝心なところです。血圧が高い状態が続くと、血管が破裂するリスクがありますよね。
そこで体はどうするか。血管が破裂しないように、周りの筋肉をギュッと固めてガードするんです。これが、マッサージに行ってもすぐ戻ってしまう「頑固な肩こり」の正体でもあるんですよ。
肩こりは単なるコリではなく、血管を守るための鎧（よろい）だったんですね。ちょっとびっくりしますよね。でも、だからこそ体には伸びしろがあるんです。
血管の状態が良くなれば、体は鎧を脱ぐ必要がなくなって、肩こりも自然と楽になっていきますよ。

3つ目は「減塩よりもカリウム」というお話です。では、具体的にどうすればいいのか。「血圧が高いなら塩を控えなさい」とよく言われますよね。
でも実は、日本人の高血圧の原因で、ナトリウム、つまり塩の摂りすぎだけが原因のかたは、思っているほど多くないんです。本当に見直すべきポイントは、「カリウム不足」なんですよ。

体の中の水分バランスや血圧は、ナトリウムとカリウムの比率で決まります。
イメージとしては、シーソーの両端に乗っているようなものなんですね。現代の食生活では、加工食品が増えて野菜が減った結果、カリウムが圧倒的に足りていません。
だから相対的にナトリウムが多くなって、血圧が上がったり、むくんだりしてしまうんです。ここまで聞いて「なるほど」と思ったかた、もう正しい方向を向いていますよ。
対策は、味気ない食事を我慢することではありません。
新鮮な野菜、つまりカリウムをしっかり摂ること。
そして、ドロドロ血の原因になる糖質や酸化した油を控えること。血管というホースがきれいになれば、心臓は無理に圧を上げる必要がなくなります。
自然と血圧は落ち着いてくるんですよ。

ここまでのお話を聞いて、「じゃあ今飲んでいる薬はやめたほうがいいの？」と思ったかたもいるかもしれません。安心してくださいね。この動画はお薬や治療を否定するものではありません。

今日お伝えしたかったのは、「数値を下げること」だけに注目するのではなく、「なぜ数値が上がっているのか」という体の声にも耳を傾けてみませんか、ということなんです。
この動画はあくまで体の仕組みを知るための「判断軸」をお渡ししているものですからね。全部を一度にやろうとしなくて大丈夫です。できることから始めれば十分ですよ。
気になることがあれば、いつでもお医者さまや私たちにお気軽に相談してくださいね。
