こんにちは！今回のテーマは「高血圧が薬で治らない本当の理由」です。
高血圧のお薬を何年も飲んでいるのに、「結局いつまで飲み続ければいいの？」と疑問に感じたこと、ありませんか？
実はここに大きな落とし穴があるんです。「治る」という言葉には2種類の意味があって、薬で言う「治る」は、皆さんが思っている「治る」とは別物なんです。
今日はこの違いを知るだけで、「何をどの順番でやればいいか」がクリアに見えてきますよ。一緒に見ていきましょうね。

結論を先にお伝えしますね。今日覚えていただきたいのは3つです。
1つ目、薬の「治る」は一時しのぎの「かんかい」。根本から良くなる「ちゆ」とは別物です。
2つ目、ちゆへの最短ルートは「骨格→ないぞう→栄養」の順番を守ること。
3つ目、迷いをなくすカギは、自分の体を「測定」して原因を確定させること。
では、さっそく本題にいってみましょう！

1つ目の「薬の"治る"は一時しのぎ」というお話です。
まず知っていただきたい言葉が2つあります。「かんかい」と「ちゆ」です。
かんかいというのは、薬で一時的に症状がおさまっているけれど、薬をやめればまた戻ってしまう状態のこと。
一方のちゆというのは、薬がなくても健康な状態がずっと続くことです。
これ、家の雨漏りに例えるとすごくわかりやすいんですよ。
雨漏りしている家で、床が濡れないようにバケツを置く。これが「かんかい」です。水は受け止められるけど、雨漏りそのものは止まっていませんよね。
一方、屋根に登って穴をふさぐ。これが「ちゆ」です。屋根を直せば、バケツはもう必要なくなる。
皆さんが「高血圧を治してほしい」と病院に行くとき、求めているのは当然「屋根の修理」、つまりちゆですよね。

ところが、高血圧をこんぽんから「ちゆ」できる薬は、まだ見つかっていません。
だから病院では、ガイドラインに沿って、一時的に数値を下げる「かんかい」のための薬、つまりバケツが処方されるんですね。
ここで問題なのが、法律上はかんかいのこともちゆのこともまとめて「治る」と表現してよいことになっている点です。
お医者さまは「バケツで水を受けますよ」という意味で「治りますよ」と言っているのに、患者さんは「屋根を直してくれるんだ」と受け取ってしまう。
このギャップが、「何年飲んでもやめられない」という悩みの正体だったんです。
でもね、ここからが希望のある話ですよ。屋根の直し方、つまりちゆに向かうための道筋がわかれば、体はちゃんと変わっていけるんです。

では、屋根はどうやって直せばいいのか。ここが2つ目のポイントです。
実は、屋根を直すにも正しい手順があるんです。
体質改善の優先順位は、1番、骨格を整える、2番、ないぞうの機能を戻す、3番、栄養を入れる、この順番です。
どんなに良い屋根材を買ってきても、骨組みがグラグラだったら乗せられませんよね。まず骨組みを直して、下地を整えて、それから初めて屋根材を張る。体もまったく同じなんです。
だから、バケツ、つまり薬だけでは屋根は直らなかったんですね。バケツは雨水を受けてくれる大切な存在だけど、屋根の修理とは別の作業なんです。

もう少し具体的にお話ししますね。ここ、知ると世界の見え方が変わるポイントです。
しせいが崩れて骨格がゆがんでいると、その中にある血管が圧迫されてしまいます。血管が圧迫されると、体は無理やり血液を押し流そうとして、血圧を上げるんです。
つまり、骨格のゆがみが高血圧の原因の一つだった。冒頭で「薬を飲んでも治らない」とお話ししましたが、その理由がここに繋がっていたんですね。
さらに、しせいが崩れるとないぞうも圧迫されます。ちょっと想像してみてください。ご飯を食べる前に、胃を3時間ギューッと押さえつけられたらどうですか？ とても消化なんてできませんよね。
ないぞうが圧迫されたままだと、せっかく良い栄養を摂ってもきゅうしゅうできない。だからバケツ、つまり薬だけ増やしても、屋根、つまりこんぽんげんいんは直らなかったんです。
この土台作りを飛ばして、例えば最新のさいせいいりょうをいきなり受けると、1回130万円以上かかることもあります。
でも、この仕組みがわかったかたはもう一歩リードしていますよ。正しい順番で骨格から整えていけば、体はちゃんと応えてくれます。これが一番コスパ良くちゆに向かう最短ルートなんです。

さあ、3つ目のポイントにいきますよ。「測定で原因を確定させる」というお話です。
骨格とないぞうを整えたら、いよいよえいようりょうほうの出番ですよね。でもここで、多くのかたがつまずく落とし穴があるんです。
ある女性のケースをご紹介します。
「自分にはあえんが足りない」と思い込んで、5年間で約90万円分のサプリを飲み続けた。なのに、数値がまったく改善しなかったんです。
なぜだと思いますか？

答えは、体の中に溜まっていた「ゆうがいきんぞく」が、ミネラルのきゅうしゅうをブロックしていたからです。
いくら栄養を入れても、邪魔者がいたら届かない。屋根を直そうとしても、ガレキが乗ったままでは作業ができないのと同じです。
つまり、ゆうがいきんぞくがいる限り、いくらサプリを飲んでも栄養は届かない。これも「薬を飲んでも治らない」の正体の一つだったんですね。
実は、日本人のまんせいてきな不調の60から70パーセントに、このゆうがいきんぞくが関わっていると言われています。
少し驚くかもしれませんが、ここに希望のタネが隠れています。邪魔者の正体がわかれば、取り除く方法もあるということですからね。
だからこそ必要なのが「測定」です。
病院で「異常なし」と言われて終わるのではなく、さいぼうレベルで「何が足りないのか」、そして「何が邪魔をしているのか」を数値で確定させる。
カーナビを想像してみてください。目的地を入れる前にまず「現在地」を確認しますよね。現在地がわからなければ、どんなに優秀なナビでもルートは出せません。測定は、まさにその「現在地の確認」なんです。

ここまでの話を聞いて、「じゃあ今飲んでいる薬は意味がないの？」と不安に思ったかたもいるかもしれません。
安心してくださいね。この動画はお薬やいりょうを否定するものではありません。
雨漏りの話を思い出してください。屋根を直しているさいちゅうに、バケツを外したらどうなりますか？ 床がびしょ濡れになりますよね。
だから、バケツ、つまり薬は屋根を直し終わるまで絶対に外してはダメなんです。お医者さまが処方されたお薬は、きちんと飲み続けてくださいね。勝手にやめちゃダメですよ〜。
今日お伝えしたかったのは、バケツで水を受けながら、同時に屋根の修理も進めていきましょう、ということなんです。これが一番賢いやり方ですからね。
気になることがあれば、いつでもお医者さまや私たちに気軽に相談してくださいね。

まず今日やること。「かんかいとちゆは違う」というお話を、ご家族や身近なかたに1つ話してみてください。人に伝えると、知識がぐっと自分のものになりますよ。
次に3日以内。鏡の前で自分のしせいをチェックしてみてください。猫背になっていないか、左右どちらかに傾いていないか。骨格のゆがみに気づくことが、屋根を直す第一歩です。
まずはこの2つから、焦らず始めてみてくださいね。

最後に、今日の3つをおさらいしましょう。
1つ目、薬の「治る」は一時しのぎの「かんかい」。根本から良くなる「ちゆ」とは別物です。
2つ目、ちゆへの最短ルートは「骨格→ないぞう→栄養」の順番を守ること。
3つ目、迷いをなくすカギは、自分の体を「測定」して原因を確定させること。
雨漏りのバケツではなく、屋根そのものを直していきましょう。正しい手順で、正しい順番で取り組めば、体はちゃんと応えてくれます。

最後に1つだけお伝えしますね。ここまで高血圧とたいしつかいぜんの仕組みをお話ししてきましたが、実は「自分の体のどこで一番エラーが起きているか」は、ひとりひとりまったく違うんです。骨格のゆがみかたも、ゆうがいきんぞくの溜まり方も、栄養のかぶそくもひとそれぞれ。
今日お話しした内容は、まさに「測定」で原因がはっきりわかることと直結しています。だからこそ、「じゃあ、私の場合はどうなっているんだろう？」と気になったかたには、測定での答え合わせをぜひ強くおすすめします。
せっかくちゆに向かう仕組みをりかいできたのに、自分の体に合わないじこりゅうの対策をして遠回りしてしまうのは、本当にもったいないですからね。
皆さんの体は、良くなる力をちゃんと持っています。あなたのたいしつに合わせた最短ルートを一緒に見つけていきましょうね。
