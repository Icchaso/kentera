こんにちは！
今回のテーマは「食事と栄養」です。「食事には気をつけているのに、なんだか疲れが取れない」「サプリも試してみたけど、いまいちてごたえがない」…そんな経験、ありませんか？でもここからが面白いところで、実はその原因にはちゃんと理由があるんです。しかも、解決の糸口もすぐそばにあるんですよ。

今日は、食事と栄養の「意外な落とし穴」をお伝えしていきます。車の例えを使って、できるだけわかりやすくお話ししますので、気軽に聞いていただけたら嬉しいです。
見終わるころには、「なるほど、だからダメだったのか」「じゃあ次はこうしよう」が自然と決まっているはずですよ。
一緒に栄養の仕組みを覗いてみましょう。

結論を先にお伝えしますね。今日覚えていただきたいのは、たった3つです。1つ目、体は「ガソリン」だけでは動かない。「オイル」が必要。
2つ目、慢性的な不調の原因は、ほとんどが「ミネラル不足」。
3つ目、「量」より「サイズ」。細胞に入れるものを選ばないと意味がない。この3つがわかると、「何を食べればいいか」の迷いがスーッと晴れていきますよ。
では、順番に見ていきましょうね。

まず、なぜ「しっかり食べているのに不調」ということが起きるのか。実は今、日本人の多くが「新型栄養失調」だと言われています。
「え？ このほうしょくの時代に栄養失調？」と思いますよね。不思議に聞こえますが、カロリーは足りすぎているのに、それを体の中で使うための「栄養」が空っぽ、という状態なんです。

そしてもう一つ、迷いやすくなる理由があります。健康情報って、本当にたくさんありますよね。「〇〇酵素がいい」「ファイトケミカルが注目」…次から次に新しいものが出てくる。
でもそれって、土台を知らないまま応用問題を解こうとしているようなものなんです。だから今日は、まず「土台」の話をします。
土台さえわかれば、どんな情報を見ても自分で優先順位をつけられるようになりますよ。
さあ、さっそく本題にいってみましょう。

はい、1つ目の「体はガソリンだけでは動かない」というお話です。栄養には大きく2つのグループがあります。
皆さんもよく聞く「タンパク質・脂質・炭水化物」、これが3大栄養素。
そこに「ビタミン・ミネラル」を加えたものが5大栄養素です。名前だけ聞くと、ちょっと教科書っぽく感じますよね。
でもこれ、車に例えるとすごくシンプルにわかるんです。

3大栄養素、つまりタンパク質・脂質・炭水化物は、車で言えば「ガソリン」と「車体そのもの」です。
車を走らせるエネルギーと、体そのものの材料ですね。一方で、ビタミンやミネラルは何にあたるかというと、「エンジンオイル」や「潤滑油」です。
エンジンの調子を整えて、スムーズに動かすための"裏方さん"なんですね。さて、ここで想像してみてください。
ガソリンは満タン。でも、オイル交換をまったくしていない車があったとします。
…どうなると思いますか？エンジンは焼き付いて、煙を上げて故障しますよね。
実はこれ、今の私たちの体で起きていることと、とてもよく似ているんです。

コンビニやスーパーの食事には、安くて美味しいガソリン、つまり3大栄養素がたっぷり入っています。
だから、ガス欠で倒れることはまずありません。でも、そのエネルギーを正しく燃やしたり、エンジンの調子を整えたりするオイル、つまりビタミンやミネラルが圧倒的に足りていないんです。カロリーという熱量はあるのに、体の機能がうまく回っていない。
これが「食べているのに調子が悪い」の正体なんですね。でもね、これって逆に言えばチャンスでもあるんですよ。
足りていないオイルを補えばいい。やるべきことがはっきりしているぶん、改善の道筋が見えやすいんです。

さあ、ここからがとても大事なお話ですよ。2つ目の「ミネラル不足」についてです。では、オイルにあたるビタミンとミネラル、どちらから優先すべきなのか。
答えは「ミネラル」です。最近は「7大栄養素」「9大栄養素」なんて言葉もありますが、慢性的な不調を根本から改善したいなら、まずミネラルを整えることが先決なんですね。
なぜかというと、3大栄養素の摂りすぎで生活習慣病になるかたはたくさんいますが、3大栄養素の不足で慢性病になるかたは、現代日本にはほとんどいません。
逆に、慢性的な不調の多くは、この「ミネラル不足」が根っこに隠れているんです。

ここで、もう一つとても大事なことをお伝えしますね。それは「時間軸」の違いです。ビタミンは、体の中での代謝が早くて、だいたい1から2週間で変動します。
だから風邪を引いたときにビタミンCを短期的に摂ると効果的、というのは理にかなっているんですね。
一方、ミネラルはどうかというと、骨や細胞の一部としてしっかり組み込まれるため、入れ替わるのに半年から、長い場合は数年かかります。
「サプリを1ヶ月飲んだけど効果がないからやめた」…実はこれ、一番もったいないパターンなんです。ダメですよ〜、途中でやめちゃ。
ミネラルは「家の基礎工事」のようなもの。
数ヶ月、年単位でコツコツ積み上げて、初めて体質という土台が変わっていきます。
焦らないこと。これが実は、体質改善の一番の近道なんですね。

さて、ここからが今日の一番おもしろいところです。3つ目は「量よりサイズ」というお話です。「じゃあミネラルをたくさん摂ればいいんですね？」…実は、そう単純でもないんです。ドラッグストアで「マグネシウム1000mg配合！」みたいなサプリ、見かけたことありませんか？
あの表示されている数字が、そのまま体に入ると思ったら…ちょっと待ってください。
ここで大事になるのが「吸収率」、もっと言えば「サイズ」のお話です。

私たちの細胞には、栄養が入るための小さな入り口があります。
「イオンチャネル」と呼ばれる穴で、そのサイズはなんと約0.7ナノメートル。ものすごく小さいんです。で、質の悪いサプリや、加工されすぎた食品に含まれるミネラルは、粒子がこの穴に対して大きすぎるんですね。わかりやすく例えてみますよ。
皆さん、ゴルフのカップの穴をイメージしてください。あの小さな穴に、ボウリングの玉を入れようとしている…。どう頑張っても入りませんよね？
これがまさに、体の中で起きていることなんです。

いくら「1000mg配合」、つまりボウリングの玉を100個用意したとしても、穴に入らなければ全部そのまま体の外に出ていくだけなんです。大事なのは、多いか少ないかじゃないんですね。
「細胞の穴を通れるサイズに、ちゃんとイオン化されているかどうか」。
これが、サプリ選びや食事で失敗しないための最大のルールです。ここまで聞いて「えっ、じゃあ今まで無駄だったの？」と思ったかた、その気づきがもう第一歩なんです。
これからは「量」ではなく「サイズ」で選ぶ。それだけで結果が変わってきますよ。

ここまでの話を聞いて、「今まで飲んでたサプリは全部意味がなかったの？」と不安になったかたもいるかもしれません。そんなことはないので、安心してくださいね。
もちろん良質なサプリもたくさんありますし、お医者さまが勧めるものにはちゃんとした理由があります。今日お伝えしたいのは、パッケージの数字だけで判断するのではなく、「ちゃんと吸収されるのかな？」という視点を一つ持ちましょう、ということなんです。
この動画は診断や治療のアドバイスではなく、あくまで「判断軸」をお渡ししているものです。
体調で気になることがあれば、いつでもお医者さまや私たちに気軽に相談してくださいね。

それでは、今日から何をすればいいか。簡単なチェックリストをお伝えしますね。まず今日やること。食事をするとき、「これにオイル、つまりビタミンやミネラルは入っているかな？」と一瞬だけ意識してみてください。それだけでOKです。

次に3日以内。海藻、だし、ごま、ナッツなど、ミネラルが豊富な食材をいつもの食事に1品だけ加えてみましょう。味噌汁にわかめを入れる、ごはんにごまをかける、それだけで十分ですよ。

そして1週間後。もしサプリを使っているかたは、裏面の表示を見て「この成分は吸収されやすい形かな？」とちょっと意識してみてください。全部を一度にやろうとしなくて大丈夫です。できることから一つずつ始めていきましょうね。

最後に、今日の内容を3つだけおさらいしましょう。1つ目、体は「ガソリン（3大栄養素）」だけでは動かない。「オイル（ビタミン・ミネラル）」がないとエンジンが焼き付いてしまう。
