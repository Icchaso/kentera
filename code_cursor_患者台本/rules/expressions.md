# 表現バリエーション集

台本を複数本作成する際、同じ言い回しの繰り返しを防ぐための表現ストックです。  
**毎回異なる表現を選び、同一台本内でも同じパターンの連続を避けること。**

## 7. 行動を促す表現（チェックリスト・CTA）

> **CTAの3原則：** ①一般論の限界を伝えて「私の場合は？」を刺激する ②測定を「答え合わせ」として提示する ③損失回避を優しく突く。「売りつけられている」ではなく「私のために用意してくれている」という安心感を生む。

### ルール1用：一般論の限界を伝える（個別化の欲求を刺激）

| # | 表現 |
|---|------|
| 1 | ここまで〇〇の仕組みをお話ししてきましたが、『自分の体のどこでエラーが起きているか』は、一人ひとりまったく違うんです。 |
| 2 | 今日お伝えしたのは全体像です。ここから先は、皆さん一人ひとりの体が『答え』を持っています。 |
| 3 | 同じ〇〇でも、原因が骨格にある方もいれば、栄養にある方もいます。だから全体像だけでは足りないんですよね。 |
| 4 | 仕組みがわかると「じゃあ私はどうなの？」と気になりますよね。実は、そこが一番大事なところなんです。 |

### ルール2用：測定＝「答え合わせ」として提示する

| # | 表現 |
|---|------|
| 1 | 今日のお話を聞いて『じゃあ、私の場合はどうなっているんだろう？』と思った方は、ぜひ自分の体の"答え合わせ"をしにきてくださいね。 |
| 2 | 動画で学んだことを、自分の体で確かめてみてください。それが測定という名の"答え合わせ"です。 |
| 3 | 〇〇の仕組みがわかった今、「じゃあ私の体では何が起きているの？」という疑問に答えてくれるのが測定なんです。 |
| 4 | 「自分の体って、実際どうなってるんだろう？」と思ったら、それがまさに答え合わせのタイミングですよ。 |

### ルール3用：損失回避を優しく突く

> **注意：** 本編で「軸」として使用済みのメタファー（例：カーナビ）がある場合、CTAでは**同じ比喩を繰り返さず、別の表現を選ぶ**こと。同じ例えの連発は「また同じ説得をされている」と視聴者に感じさせてしまう。

| # | 表現 |
|---|------|
| 1 | 自己流で遠回りしたり、せっかくの努力を無駄にしないためにも、まずは現在地を確定させておくのが一番の近道ですよ。 |
| 2 | 頑張る方向を間違えると、時間もお金ももったいないですよね。だからこそ、最初に正しい現在地を知っておくことが大切なんです。 |
| 3 | カーナビと同じで、現在地がわからなければ正しいルートは出せません。まずは今の状態をはっきりさせておきましょう。 |
| 4 | せっかく仕組みを理解したのに、自分の体に合わないことをやっていたらもったいないですよね。まずは答え合わせからですよ。 |

| 1 | あなたの体質に合わせた最短ルートを一緒に見つけていきましょうね。 |

| 3 | 今日「へぇ」と思ったこと、ぜひご家族にも話してみてください。人に伝えると、知識がぐっと自分のものになりますよ。 |

## 9. 比喩・例え（動画の「軸」として活用）

> **核心ルール：** 1つの動画につき強力なメタファーは**1〜2個に絞る**。分散させず、動画全体の「軸」として繰り返し使い、視聴者の記憶に刻む。

### メタファーの選び方
- テーマの核心を1つの例えで貫けるものを選ぶ
- 優れたメタファーは**複数の役割を持たせられる**（説明＋リスクヘッジ＋行動の動機づけ等）
- 例：「雨漏りのバケツ＝薬（一時しのぎ）、屋根の修理＝治癒（根本治療）」→ 対症療法 vs 根本治療の説明だけでなく、「薬は勝手にやめてはいけない（バケツは今も必要）」というリスクヘッジにも使える

### メタファーの配置
1. **導入〜本編序盤**で最初に提示する
2. **本編中盤**で繰り返し参照し、各ポイントを同じ例えで説明する
3. **まとめ**でもう一度使い、記憶に定着させる
- この「提示→参照→定着」の3段階で、視聴者の頭に共通言語として残す

### 比喩パターンストック

| # | 比喩パターン | 用途例 |
|---|-------------|--------|
| 1 | 車（ガソリン・オイル・エンジン） | 栄養・エネルギー・代謝の関係 |
| 2 | 家（土台・基礎工事・雨漏り） | 体質改善、根本 vs 対症療法 |
| 3 | カーナビ（現在地・目的地・ルート） | 測定の必要性、治療計画 |
| 4 | 銀行口座・貯金 | 栄養の蓄積、長期の積み上げ |
| 5 | 道路・渋滞 | 血流、流れの滞り |
| 6 | スイッチ・電気 | 体のスイッチ、活性化 |
| 7 | 料理・レシピ | バランス、組み合わせ |
| 8 | チーム・役割分担 | 臓器・細胞の協力 |
| 9 | 水やり・植物 | 継続、適量、タイミング |
| 10 | 交通整理・信号 | 優先順位、流れのコントロール |
| 11 | ゴルフカップとボウリングの玉 | 吸収率、サイズ・形の重要性 |
| 12 | 家づくりの順番（基礎→柱→屋根） | 治療の優先順位、飛ばせない手順 |

### 運用ルール
- 1動画で使うのは**1〜2個まで**。それ以外の例えは使わない
- 選んだメタファーを**動画全体の共通言語**にする（冒頭・本編・まとめで繰り返す）
- 比喩を使う場合は「〇〇で例えると」と一言添えると、聞き手が切り替わりやすい

### 同じ比喩を複数台本で使うとき
- どうしても同じ例えが一番わかりやすい場合は、**「前にも使ったけど」と断りを入れて使ってよい**
- 例：「以前の動画でもお伝えしましたが、車のオイルの例えが一番わかりやすいので、改めてお話ししますね」

## 使い方のルール

1. **同一台本内で同じ表現を2回以上使わない**
2. **直近で生成した台本と表現がかぶらないよう、毎回異なる番号の表現を選ぶ**
3. 上記はあくまでストック。**テーマに合わせて自然にアレンジしてよい**（丸写しではなく、文脈に馴染む形に調整する）
4. ストックにない新しい言い回しを生み出すのも積極的に推奨する
5. **1つのカテゴリ内で偏りが出ないよう、幅広く使い分ける**
6. **メタファーは1動画1〜2個に絞り、動画全体の「軸」として繰り返す**（分散させない）
7. **CTA表現は提案型を基本**（押しつける言い切りは避け、理由と選択肢で自然に行動を促す）
