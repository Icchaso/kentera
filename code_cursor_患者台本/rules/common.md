# 共通ルール

患者向け解説動画の台本生成時の基本基準です。（10〜15分想定）

## 台本の絶対ルール

- **1本＝1メッセージ＋1CTA**（欲張るほど弱くなる）
- **結論→理由→手順→例→まとめ**の順（視聴維持が伸びる）
- **3点構成**にする（人は3つまでしか覚えない）
- 不安を煽らない（特に医療領域は"安心設計"がCVに直結）
- **理解と行動は分けて設計する**（下記「二層構造」）。行動レベルでは継続可能性を優先し、禁止だけで終わらせない

---

## 構成4原則

### 原則1：一本の線で繋ぐ（伏線の回収）

台本の最重要原則。冒頭のフック（悩み）→ 中盤のメカニズム解説 → CTAまで、**因果関係の一本線**で繋ぐ。テーマが分断されてはならない。

- 冒頭で提示した悩みは、中盤で「**だからその悩みが起きているんです**」と必ず回収する
- 視聴者に「なるほど、全て繋がっていたのか！」というアハ体験を与える
- 各ポイント間の因果の鎖を明示する（例：「骨格が歪む → 血管が圧迫される → 血圧が上がる」）
- 最後のCTAも本線の延長線上に置く（唐突な誘導にしない）

### 原則2：「教育」に尺を使う（説得は最小限）

KENTERAに参加している時点で一定の信頼がある。冒頭の長い共感・前置き・動機づけは不要。

- **最初の1分でパラダイムシフト（新しい視点・常識の転換）を与える**ことが勝負
- 共感・前置きは最短にとどめ、最速でコアの教育コンテンツに入る
- 「説得する時間」を「教える時間」に置き換える
- 「なぜ迷うのか」「情報が多すぎる」等の一般論は省くか、一言で済ませる

### 原則3：メタファーは「軸」にする

1つの動画につき強力なメタファーは**1〜2個に絞り**、動画全体の軸として繰り返し使う。

- 分散した例え話を並べるのではなく、1つの例えを**共通言語**として貫く
- 優れたメタファーは複数の役割を持たせる
  - 例：「バケツ＝薬（一時しのぎ）」「屋根の修理＝治癒（根本治療）」→ これだけで「薬は勝手にやめてはいけない（バケツは今も必要）」というリスクヘッジの補強にもなる
- メタファーを最初に提示し、本編中に繰り返し参照することで記憶に刻む
- **同じフレーズの連呼禁止：** メタファーを「軸」にすることと、同じフレーズをコピペのように何度も繰り返すことは別物。同じ比喩フレーズ（例：「カーナビと同じで現在地が〜」）を2回以上そのまま使うと、視聴者は「また同じ説得をされている」と察知し心が離れる。参照する場合は表現を変えるか、別の角度から言い換えること

### 原則4：CTAは「自然に行きたくなる」設計にする

一律の強い命令口調は押しつけになりやすい。**医学的根拠を示したうえで、視聴者が"自分から"動きたくなる理由づけ**で導く。

以下の3つのルールで設計する：

1. **一般論の限界を素直に伝える**（主役を視聴者に切り替える）
   - 本編で仕組みをたっぷり伝えたあと、「でも、これはあくまで全体像です」と引く
   - 「じゃあ、私の場合はどうなの？」という**個別化（パーソナライズ）の欲求**を刺激する
   - 人は一般論には感心するだけだが、「自分ごと」になると自ら答えを探しに行く

2. **「測定」を「答え合わせ」に変換する**
   - 「測定に行きましょう」→「テストを受けさせられる」というストレスになる
   - そうではなく、動画で学んだ知識を自分の体に当てはめる**「答え合わせの場」**として提示する

3. **「損をしたくない心理」を優しく突く**（損失回避）
   - 「行くと健康になりますよ」（メリット訴求）より、「行かないと努力が無駄になるかも」（損失回避）のほうが自然で強力な動機
   - ただし脅しにならないよう、柔らかく添える

→ 「売りつけられている」ではなく「私のために用意してくれている」という安心感を生む

### 「日常アクション」と「測定CTA」の完全分離（必須）

- 台本終盤の「今日からのアクション（チェックリスト）」と「CTA（測定の案内）」は**必ず別のスライドに分離する**
- **チェックリストには「視聴者が今日から自力で実践できる日常の行動」だけ**を入れる（例：家族に話す、姿勢をチェックする、食事を見直す等）
- **測定の案内はチェックリストに絶対に入れない**。測定はCTAスライドでのみ扱う
- チェックリストとCTAの両方で測定に触れると「二重の売り込み」になり、「自然に行きたくなる」設計が崩れる

### CTA運用の大原則

> **強引になるくらいなら、潔くオファーは捨てる。測定会のCTAは、本文の内容と自然に結びつく場合だけしっかり行う。**

- すべての動画で同じ強さのCTAを入れない。**本文の内容と測定会の結びつきの強さ**で Tier 1（しっかり推奨）と Tier 2（さらっと一言）を使い分ける
- 結びつきが強い場合：「強くおすすめします」くらいの温度感でOK
- 結びつきが薄い場合：「気になる方はチェックしてみてくださいね」程度で済ませる。強引にくっつけない
- **迷ったらTier 2**にする（しつこくなるリスクを避ける）
- 視聴者が「自然と行きたくなる」構成を最優先する
- 詳細は `patient.md` のCTA Tier設計を参照

## 結論と本文の対応（必須）

- **結論で言った数・内容と、本文が1対1で対応すること**
- 「結論は3つです：A・B・C」と言ったら、本文の構成は次のように揃える：
  1. **1つ目のAについて** … 冒頭で「1つ目」と明示し、Aを説明
  2. **2つ目のBについて** … 「2つ目」と明示し、Bを説明
  3. **3つ目はCについて** … 「3つ目」と明示し、Cを説明
- 結論のA・B・Cは**誰でもわかりやすい表現**にする（専門用語を避け、日常の言葉で言い切る）
- 本文での説明には**比喩（例：車のガソリンとオイル、ゴルフカップとボウリングの玉など）**を使って、イメージしやすくする

## トーン：明るく前向きな文体

台本全体を通して「明るさ」と「希望」を基調とする。

### 問題提起→希望のセット（必須パターン）
- ネガティブな事実や課題を提示したあとは、**必ず前向きなフォロー**を添える
- 「でも大丈夫」「裏を返せば〜」「希望のある話をしますね」「改善が期待できる」等
- 問題で終わらせず、**その先の可能性**を示して締める
- 例：✕「これが詰まるの正体です。」→ ◯「これが詰まるの正体なんですね。でも裏を返せば、柔らかさを取り戻せれば流れも良くなるということなんです。」

### ワクワク感を持たせる
- 仕組みの解説は「すごい秘密」「面白い仕組み」など、好奇心をくすぐる表現を使う
- 導入や締めで「楽しんでいってくださいね」「一緒に見ていきましょう」等、参加感を出す

### ネガティブ表現を和らげる
- 「怖い」→「ドキッとする」
- 「諦めかけていませんか」→「思っていませんか」
- 「仕方ない」→「理由がある」
- 直接的に不安を連想させる言葉はワンクッション置いて柔らかくする

### 締めは必ず明るく
- 各セクション・スライドの結びは前向きなメッセージで終わる
- 台本の最後は「皆さんの体は良くなる力を持っています」のような希望あるメッセージで閉じる

### NG表現
- 「〜だけ持ち帰ってください」 → 一方的・押しつけ的な印象になるため使わない
- 「〜してください」の連発 → 「〜してみてくださいね」「〜していきましょう」等に言い換える

### 表現の重複回避（複数台本対応）
- **`rules/expressions.md`** に8カテゴリの表現バリエーション集を用意している。台本生成時に必ず参照する
- 同一台本内で同じ言い回しを2回以上使わない
- 直近で生成した台本と表現がかぶらないよう、毎回異なるパターンを選ぶ
- ストックを丸写しせず、テーマに合わせて自然にアレンジする
- ストックにない新しい言い回しを積極的に生み出すことも推奨する

## 台本の先頭に必ず置く「3行サマリー」

毎回書く：

- **この動画のゴール：**＿＿＿＿
- **視聴後にできること：**＿＿＿＿（行動レベル）
- **次の一手（CTA）：**＿＿＿＿（1つだけ）

## 前提

- 入力する元テキストは、医学的根拠に基づくデータ・エビデンスを前提としている

## 医療情報の取り扱い

- 元テキストの医学的データは正確に反映する
- 推測や創作は行わない。不明点は「医師にご確認ください」等で補足
- 適切な箇所で**相談の促し**を入れる  
  例：「このような異変を感じる場合は、すぐに我々か医師にご相談ください」

## 禁忌事項

- 個人を特定できる情報は台本に含めない
- 過度に不安を煽る表現は使わない
- 他社製品・治療法の disparage は行わない

## 構成の目安

- 1スライドあたりのナレーションは 30〜90 秒を目安（読みやすさ・理解しやすさ）
- 区切りが良いタイミングでスライドを分ける

## 現実行動デザイン（必須）

日常習慣に関する提案（食事・睡眠・飲酒・運動など）は、視聴者が実際に続けられる形で設計する。

### 二層構造（理解と行動を分ける）

医学的根拠に基づき、**「それがなぜ悪いのか」「何がNGなのか」ははっきり伝える**。曖昧にすると誤解や自己流が増える。  
一方で、**人はいきなり完全禁止には移行しにくい**ので、理解のあとに続く**具体的な一歩**は段階・代替・最低ラインで支える。

| レイヤー | 目的 | やること |
|---|---|---|
| **① 理解（境界）** | 悪いことを悪いと理解してもらう | 根拠を添えて「やってはいけない／避けるべき」を明言する |
| **② 行動（実行）** | 続けられる形に落とす | 段階化・最低ライン・代替案・再開ルールを必ずセットにする |

「ダメ」と言うことと、「今日からの一歩」を提案することは矛盾しない。**①で線を引き、②で越えないように手立てを渡す**イメージ。

### 原則

- **①のあとに②を必ず置く**：境界だけで終わらせない（「だからこう減らしていこう」まで言う）
- **段階的に進める**：いきなりゼロを唯一の正解にしない。1週目→2週目のようにステップ化する
- **最低ラインを明示する**：守れたら合格の小ルール（時間・量・頻度）を先に示す
- **代替案をセットで出す**：同時に実行できる代替行動（飲み物、行動、習慣）を具体的に提示する
- **失敗前提で語る**：できない日があっても翌日戻せばよい、という再開ルールを必ず添える

### 使ってよい言い方 / 避ける言い方

- 使ってよい：根拠付きで「これは避けた方がよいです」と線を引く → 続けて「まずはここまでで大丈夫」「できる日からでOK」
- 避ける：**根拠なしの命令**だけで終わること、**説明なく「絶対」「今すぐゼロ」だけを突きつける**こと、**守れない人を責める**言い方（「意味がない」など）

### 夜のビールを扱うときの運用例

- **晩酌は可、寝酒は不可**：就寝3時間前までに飲み終える
- **同量の水をセット**：ビール1杯に対して水1杯を必ず入れる
- **メリハリ運用**：毎日固定ではなく、翌日に備える日を休肝日にする

> 注意：服薬中・既往歴・医師の指示などで飲酒自体が不適切なケースは、必ず「医師に確認」を添えること。
