## 1. KENTERAとは何か

**KENTERA（健康寺子屋）** は、患者に「正しい健康知識」と「判断軸」を渡し、最終的に **測定会 → 治療院のバックエンド成約** につなげる **教育・導線システム** です。

### 治療家にとっての一言まとめ

> 「患者を教育して、納得した状態で測定会に来てもらい、治療院のバックエンド商品が **自然に売れる仕組み**。」

### ポイント

- 患者向けサービスに見えるが、本質は **治療院の売上最大化モデル**
- 「説明 → 提案 → 断られる」という従来パターンを根本から変える
- 先生が毎回ゼロから説明する必要がなくなり、**施術に集中できる環境** を作る

---

## 2. なぜ今のやり方では限界があるのか

### 従来のパターン

```
説明して → 提案して → 「ちょっと考えます」 → 離脱
```

これは患者さんが悪いのではなく、**受け取る準備ができていないまま提案されている** から起きる構造的な問題です。

### 具体的な課題

| 課題 | 原因 |
|------|------|
| 高単価メニューが成約しない | 患者が「なぜ必要か」を理解していない |
| 数回来て終わる | 痛みが取れた時点で通院の目的がなくなる |
| 説明に時間を取られる | 毎回ゼロから同じ説明を繰り返している |
| 紹介が生まれない | 患者が「何がよかったか」を言語化できない |
| 稼働率が上がらない | 新規に依存し、継続・リピートの仕組みがない |

### KENTERAモデルが解決すること

```
教育して → 測定して → 納得させてから → 提案する
```

**提案の前に「教育」と「測定（数値化）」を置くことで、患者の意識段階を引き上げ、提案が「自分に必要なこと」として届くようにする。**

## 3. KENTERAモデルの4ステップ

```
① 教育  →  ② 測定  →  ③ バックエンド成約  →  ④ LTV向上
```

この **順番** が極めて重要です。いきなり②や③から入ると、どれだけいいサービスを提案しても「ふーん」で終わります。

| ステップ | やること | 使うもの | 治療家の役割 |
|---------|---------|---------|------------|
| ① 教育 | 患者に健康の基礎知識・判断軸を渡す | KENTERA（FANTS動画学習アプリ） | 患者をKENTERAに案内するだけ |
| ② 測定 | 患者の体の状態を数値で可視化する | オリゴスキャン（測定会） | 測定会を開催・運営する |
| ③ 成約 | 数値に基づいた改善メニューを提案する | 院独自のバックエンドメニュー | 結果説明 → 提案（押し売りゼロ） |
| ④ LTV | 継続通院・紹介を仕組み化する | 定期測定・フォロー・栄養療法 | 月次KPIで改善サイクルを回す |

## 4. ステップ1：教育（KENTERA / FANTS）

### KENTERAの役割

KENTERAは患者向けの **動画学習アプリ**（FANTSプラットフォーム上）です。体の仕組みや健康の基礎を、患者が **自分のペースで** 学べます。

### 治療家にとっての最大のメリット

> **「先生が全部教えなくていい。」**

- KENTERAが代わりに教えてくれるので、患者は「なぜこのケアが必要なのか」を **理解した状態** で来院する
- 先生は「それを踏まえて、あなたの場合は…」という **個別の提案** に集中できる
- 毎回ゼロから説明する工数がなくなる

### 患者の学習ステップ

教育コンテンツは体系的に構成されています：

| シリーズ | 内容 | 本数 |
|---------|------|------|
| **A：導入** | KENTERAの使い方、測定会の紹介、注意事項 | 5本 |
| **B：基礎9ステップ** | 血液・栄養・血糖・免疫・炎症・有害金属・サプリ・睡眠 | 9本（完成済み） |
| **C：症例別** | 高血圧・糖尿病・アレルギー・ダイエット・睡眠改善 等 | 16本 |

#### B：基礎9ステップの構成

患者が「体の仕組み」を体系的に理解するための幹となるシリーズ：

| STEP | 領域 | 動画 |
|------|------|------|
| STEP1 | 血液 × 栄養 × 血糖 | B01：血液系の全体像 / B02：現代食の落とし穴 / B03：糖尿病・高血糖 |
| STEP2 | 免疫 × 炎症 × 血管 | B04：免疫系の全体像 / B05：アレルギー・アトピー / B06：高血圧 |
| STEP3 | 最適化 | B07：有害金属 / B08：サプリのルール / B09：睡眠改善 |

### 院内でのKENTERA案内

- **患者への紹介トーク**（30秒・2分・5分の台本を用意）
- **よくある抵抗への返し**（「怪しい」「面倒」「忙しい」等）
- **院内導線**（POP・QRコード・受付・LINE文面テンプレート）
- **既存患者の掘り起こし**（休眠患者への再アプローチ）

### 教育コンテンツの設計思想

患者向け動画は以下の原則で作られています：

1. **1本＝1メッセージ＋1CTA**（欲張るほど弱くなる）
2. **結論→理由→手順→例→まとめ** の順（視聴維持が伸びる）
3. **3点構成**（人は3つまでしか覚えない）
4. **不安を煽らない**（医療領域は "安心設計" がCVに直結する）
5. **測定会への誘導は自然な流れで**（押し売り感ゼロの「答え合わせ」として提示）

## 5. ステップ2：測定（オリゴスキャン）

### オリゴスキャンとは

体内の **ミネラルバランス** と **有害金属の蓄積状態** を数値で可視化できる測定機器です。

### なぜ測定が重要か

> **「知識が、納得に変わる瞬間。」**

- KENTERAで「有害金属が慢性疾患に関係している」と学んだ患者が、**自分の数値を見る**
- 「あ、自分もアルミが高い」「鉄が足りてない」と **自分ごとになる**
- この瞬間に「一般論」が「自分の問題」に変わる

### 数字で見る重要性

- **有害金属は日本人の慢性疾患の60〜70%に関与** している
- 「異常なし」で終わらず、**具体的な改善のきっかけ** が見つかる
- 適切な栄養療法を取り入れれば、有害金属の排出も改善可能（例：カドミウムの生物学的半減期38年 → 適切な栄養療法で1ヶ月1週間で25%排出した事例も）

### 治療家にとっての測定の意味

| 項目 | 効果 |
|------|------|
| 成約率の向上 | 数値という客観的根拠があるため、提案に説得力が出る |
| 説明工数の削減 | 「なぜ必要か」を数値が証明してくれる |
| 患者の意識段階を上げる | 無関心（60%）の層を「考えている」に引き上げる |
| 差別化 | 他院にはない「数値化」という武器 |
| 継続理由の明確化 | 定期測定で「変化が見える」→ 継続動機になる |

## 6. ステップ3：バックエンド成約

### 教育＋測定を経た後の提案は「売り込み」ではない

教育と測定を経た患者にとって、先生からの提案は **「自分に必要な改善策」** として届きます。

### 成約の流れ

```
測定結果の説明
  ↓
「あなたの場合、ここが優先です」（数値に基づく優先順位の提示）
  ↓
改善メニューの提案（バックエンド）
  ↓
成約 or 持ち帰り → フォロー（3日以内）
```

### 結果説明のポイント

- 数値で **「今の状態」** を可視化する
- **「何から始めればいいか」** の優先順位を明確にする
- バックエンド提案は押し売りではなく、**「次の一歩の提案」** として設計する
- 患者がFANTSで学習済みなら、説明は大幅に省力化できる

### 価格帯の目安

| 価格帯 | メニュー例 | 特徴 |
|--------|----------|------|
| 約5万円 | 一般治療（肩こり・腰痛等） | 市販薬でごまかせる領域 |
| 約10万円 | 痛風・頭痛・坐骨神経痛等 | 医薬品が必要な領域 |
| 高単価 | 慢性疾患・体質改善・悩み解決 | 総合病院レベル、治癒を目指す領域 |

## 7. ステップ4：LTV向上（継続・紹介）

### 継続の仕組み

- **定期測定**：数値の変化が見えることで継続動機が生まれる
- **栄養療法**：長期化しやすく満足度が高い（＝高付加価値で長期通院）
- **KENTERA継続学習**：770円/月の月額コンテンツで接触頻度を維持
- **月次KPI**：院別の数字の見方・改善会の回し方

### 紹介が生まれる理由

教育→測定→成約を経た患者は：

1. **「なぜ良くなったか」を言語化できる**（KENTERAで学んでいるから）
2. **数値という客観的な証拠がある**（「こんなに変わった」と見せられる）
3. **自分が納得しているから、自信を持って勧められる**

## 8. 治療の優先順位モデル

KENTERAが推奨する、コストパフォーマンスの高い治療順序：

```
① 筋骨格整体  →  ② 内臓整体  →  ③ 栄養療法  →  ④ 医療
```

### なぜこの順番なのか

| 順序 | 治療 | 理由 |
|------|------|------|
| ① | 筋骨格整体 | 姿勢の崩れ → 負荷集中 → 痛みの根本。まずここを正す |
| ② | 内臓整体 | 筋骨格が歪む → 内臓が圧迫される → 機能低下。整体後に内臓を戻す |
| ③ | 栄養療法 | 内臓が機能低下 → 同じ食事でも栄養吸収が悪い。内臓を整えた上で栄養を補う |
| ④ | 医療 | 慢性疾患の治癒を医療で行うと非常に高額。①〜③で改善しない場合の最終手段 |

### 内臓圧迫のイメージ

> 食事の3時間前から、ずっと胃をグーッと押され続けた状態で「いつも通り食べてください」と言われたら、いつも通り食べられるでしょうか？ ── 筋骨格が歪んでいるということは、まさにそれが内臓で起きているということです。

### クロスセルの考え方

多くの治療院は「筋骨格の整体（痛みを取る施術）」だけで止まっています。しかし：

- 筋骨格がねじれている → 内臓も圧迫されている
- 内臓が機能低下 → 栄養の吸収が悪くなっている

つまり **痛みの先にある「体質改善」のニーズ** は、ほぼすべての患者に潜在的に存在します。これを **提案するだけで** 売上が変わります。無理に売りつけるのではなく、**メニューとして用意し、案内する** だけです。

## 9. 意識段階モデル（オーツの法則）

患者の心理状態を理解することが、成約率を左右します。

```
無知（27%）→ 無関心（60%）→ 考えている（10%）→ 渇望（3%）
```

| 段階 | 割合 | 患者の状態 | 必要な対応 |
|------|------|----------|----------|
| **無知** | 27% | 自分の問題を認識していない | 認知させる。気づきのきっかけを与える |
| **無関心** | 60% | 問題は知っているが解決する気がない | **測定で数値化して自分ごとにする** |
| **考えている** | 10% | 解決策を探している | 「うちならこうできます」と優位性を伝える |
| **渇望** | 3% | 今すぐ解決したい | 「今決めた方がいい理由」を伝える |

### 重要な気づき

> **メニューの素晴らしさを伝えて響くのは、全体の13%だけ。**

残り87%の患者は、まず **意識の段階を上げる** ことが必要です。それを実現するのが「教育（KENTERA）」と「測定（オリゴスキャン）」の組み合わせです。

## 10. コンセプトチェンジ：痛みのケアから体質改善へ

### 従来のコンセプト

```
痛みのケア（寛解 = 一時的に症状が緩和するが、また戻る）
```

### 新しいコンセプト

```
体質改善（治癒 = 病気の状態から健康に戻り、再発しない状態）
```

### 寛解と治癒の違い

| | 寛解 | 治癒 |
|---|------|------|
| 定義 | 一時的に病気の状態が緩和される | 健康な状態に戻り、再発しない |
| 短期的に見ると | 改善したように見える | 改善したように見える |
| 長期的に見ると | **再発する** | **戻らない** |
| 患者の行動 | 数回来て「治った」と思い離脱 | 治癒まで継続通院する |

### なぜコンセプトチェンジで売上が上がるのか

- 患者の目的が「痛みを止める」から「体質を改善する」に変わる
- 同じ施術でも、**目的が変われば価値が変わる**（目的と価値の法則）
- 体質改善は長期間のケアが必要 → **継続通院** につながる
- 実績：コンセプトチェンジにより **1日で267万円の売上** を達成した院もある

## 11. メニュー拡充の考え方

### アップセル vs クロスセル

治療院業界では **クロスセルで分単価を上げる** のが効果的です。

| 方法 | 内容 | 治療院での例 |
|------|------|------------|
| アップセル | 上位プランの販売 | 回数券10回→20回（※分単価は下がる） |
| **クロスセル** | **関連サービスの追加販売** | **筋骨格整体 + 内臓整体 + 栄養療法** |

### クロスセルのメニュー展開

```
既存メニュー（筋骨格整体）
  ＋ 内臓整体（筋骨格の歪みによる内臓圧迫の改善）
  ＋ 栄養療法（内臓機能改善後の栄養バランス是正）
  ＋ 医療提携（必要に応じた医療機関の紹介）
```

### 販売しやすいメニュー例：痛風（高尿酸血症）

- 推定患者数：**1,000万人以上**（30代男性の約30%）
- 体内酸性度の改善（クエン酸等）で改善可能
- 栄養療法として長期的に通っていただける

## 13. 患者教育コンテンツの体系

### 動画の設計思想

患者は「理解したい」より先に、**安心したい／迷いを止めたい／次に何をすればいいか知りたい** が本音です。

動画は以下のゴールで設計されています：

1. **わからなかったことがわかるようになる**（理解のハードルを下げる）
2. **日々の具体アクションが決まる**（「明日から何をするか」が明確になる）
3. **自然にもっと学びたいと思ってもらう**（興味・好奇心が湧く流れ）
4. **無理なく続けられる**（完璧主義ではなく再現できる最小単位の行動）

### 動画の基本構成（10〜15分）

| 時間帯 | パート | 内容 |
|--------|-------|------|
| 0:00-0:15 | 共感フック | 悩みを代弁し、すぐ本題へ |
| 0:15-0:45 | パラダイムシフト | 常識を覆す新しい視点を投下 |
| 0:45-1:00 | 結論先出し | 「今日のポイントは3つ」 |
| 1:00-8:00 | 本編 | メカニズム解説＋伏線回収（3点） |
| 8:00-10:00 | 不安ゼロ設計 | Q&A・注意・医療との違い |
| 10:00-12:00 | アクション | 今日からできるチェックリスト |
| 12:00-14:00 | まとめ＋CTA | 復唱→自然な測定会への誘導→希望のメッセージ |

### CTA（行動喚起）の設計

測定会への誘導は、**「売りつけられている」ではなく「私のために用意してくれている」** という安心感で設計：

1. **一般論の限界を伝える**：「でも、これはあくまで全体像です」→ 「じゃあ私の場合は？」を刺激
2. **測定＝「答え合わせ」**：テストではなく、学んだことの確認の場として提示
3. **損失回避を優しく突く**：「せっかくの努力を無駄にしないために」

## 14. 測定会の設計と運営

### 測定会のフェーズ

| フェーズ | 内容 |
|---------|------|
| **事前準備** | 会場・機材・案内 |
| **集客** | 院内告知 × 既存患者 × FANTS告知の分担 |
| **当日運営** | 受付 → 雰囲気づくり → 測定実施 |
| **結果説明** | 数値の見せ方・不安解消・優先順位の明確化 |
| **バックエンド提案** | 数値に基づく「次の一歩」の提案 |
| **フォロー** | 当日〜3日以内のフォロー（次回予約・継続の決め方） |

### 測定会の当日フロー

```
受付・雰囲気づくり
  ↓
測定実施（オリゴスキャン）
  ↓
数値の説明（不安解消 + 優先順位の明確化）
  ↓
バックエンド提案（「次の一歩」として）
  ↓
成約 or 持ち帰り
  ↓
フォロー（3日以内）
```

### 成約率を上げるポイント

- **FANTSで学習済みの患者は説明が省力化できる**（教育→測定の順番が効く）
- 数値という客観的な根拠があるから、**患者自身が「必要だ」と判断** できる
- バックエンドは押し売りではなく、**「あなたの数値を見ると、ここから始めるのがベストです」** という提案
- クレーム予防の不安解消トークを事前に準備しておく

## 15. 初診対応とクロージング

### 問診・ヒアリング・カウンセリングの違い

多くの院ではこの3つが混同されています。明確に分けることで成約率が変わります：

| | 問診 | ヒアリング | カウンセリング |
|---|------|----------|--------------|
| **時間軸** | 現在・過去 | 現在・過去 | **未来** |
| **内容** | 体の状態 | 心の状態 | ゴール設定 |
| **施術者の立場** | 理解者 | 理解者 | **専門家** |
| **治療家側の目的** | 体の情報を知る | 来院のきっかけ、前院を離れた理由を知る | 認識・行動を改めるきっかけを作る |

### クロージングのタイミング（重要）

**一般的なやり方（非推奨）：**
```
問診・ヒアリング → メニュー案内 → 金額提示 → クロージング
```
→ 金額の「YES/NO」になり、大抵の人は「お金を払いたくない」が勝つ

**KENTERAが推奨するやり方：**
```
問診・ヒアリング → カウンセリング → クロージング → メニュー案内 → 金額提示
```
→ 「どういう方向に進みたいか」をクロージングしてからメニューを提案する

## 16. 導入後のKPIと改善サイクル

### 追うべき指標

| KPI | 見るポイント |
|-----|------------|
| KENTERA登録数 | 院内案内が機能しているか |
| FANTS視聴完了率 | コンテンツが患者に届いているか |
| 測定会参加率 | 教育→測定の導線が機能しているか |
| バックエンド成約率 | 結果説明→提案の流れが機能しているか |
| 継続率（3ヶ月・6ヶ月） | LTV向上の仕組みが機能しているか |
| 紹介数 | 患者満足度の指標 |
| 分単価 | クロスセルが機能しているか |

### 改善サイクル

```
月次KPI確認
  ↓
ボトルネック特定（どこで患者が離脱しているか）
  ↓
改善施策の実施
  ↓
翌月の数値で検証
```

## 17. よくある反論と回答

| 反論 | 回答 |
|------|------|
| **「忙しくて運用できない」** | このモデルは工数削減が前提の設計。患者教育はKENTERAが代わりにやるので、むしろ今より楽になる |
| **「患者がついてこれるか心配」** | 患者教育はFANTSが代わりにやる。院の負担はほぼゼロ。動画は専門用語を使わず、わかりやすく設計されている |
| **「費用対効果が見えない」** | 測定会1回で成約が出た事例がある。コンセプトチェンジで1日267万円の売上実績もある |
| **「今は時期じゃない」** | 今動かない院ほど、半年後に差がつく。先に仕組みを作った院が患者を取る |
| **「怪しくないか」** | オリゴスキャンは医学的エビデンスに基づく測定。KENTERAのコンテンツも医学的根拠を前提に制作している |

## 18. まとめ：治療家にとってのKENTERA

### KENTERAは「売上OS」

KENTERAは単なる患者向けアプリではなく、治療院の **売上を構造的に最大化する仕組み（OS）** です。

### 4ステップの力

```
教育で土台を作る → 測定で納得を作る → 提案が「自分に必要なこと」として届く → 継続と紹介が生まれる
```

### 導入によって変わること

| Before | After |
|--------|-------|
| 毎回ゼロから説明する | KENTERAが教育してくれる |
| 提案しても「考えます」で終わる | 数値に基づく提案が自然に通る |
| 痛みが取れたら離脱 | 体質改善のため継続通院する |
| 新規に依存する | 継続と紹介で安定する |
| 低単価メニューしか売れない | クロスセルで高単価メニューが動く |
| 説明に時間を取られる | 施術に集中できる |

### 最初の一歩

1. **自院の患者導線を紙に書き出す**（初診から継続まで）
2. **「教育が入っていない箇所」と「測定で変わりそうな箇所」を探す**
3. **4つのステップのどこが欠けているかを確認する**

現状が把握できれば、何をすればいいかは自然に見えてきます。
