# 表現バリエーション集（治療家向け専用）

台本を複数本作成する際、同じ言い回しの繰り返しを防ぐための表現ストックです。  
**毎回異なる表現を選び、同一台本内でも同じパターンの連続を避けること。**

---

|---|------|

| 6 | でもここからが面白いところで、実は解決の糸口もすぐそばにあるんですよ。 |
| 7 | 一見大変そうに聞こえますが、やることはとてもシンプルなんです。 |
| 8 | これって逆に言えば、知っている院だけが差をつけられるチャンスでもあるんですよ。 |

## 9. 比喩・例え（仕組みの説明に活用）

| # | 比喩パターン | 用途例 |
|---|-------------|--------|
| 1 | 車（ガソリン・オイル・エンジン） | 栄養・エネルギー・代謝の関係 |
| 2 | 家（土台・基礎工事） | 体質改善、根本からの整え方 |
| 3 | ゴルフカップとボウリングの玉 | 吸収率、サイズ・形の重要性 |
| 4 | 銀行口座・貯金 | 栄養の蓄積、長期の積み上げ |
| 5 | 道路・渋滞 | 血流、流れの滞り |
| 6 | スイッチ・電気 | 体のスイッチ、活性化 |
| 7 | 料理・レシピ | バランス、組み合わせ |
| 8 | チーム・役割分担 | 臓器・細胞の協力 |
| 9 | 水やり・植物 | 継続、適量、タイミング |
| 10 | 交通整理・信号 | 優先順位、流れのコントロール |

- テーマに合う比喩を選び、核心の説明に絡める
- 比喩を使う場合は「〇〇で例えると」と一言添える
- 直近の台本で使った比喩は避け、別のパターンを選ぶ

---

## 使い方のルール

1. **同一台本内で同じ表現を2回以上使わない**
2. **直近で生成した台本と表現がかぶらないよう、毎回異なる番号の表現を選ぶ**
3. 上記はあくまでストック。**テーマに合わせて自然にアレンジしてよい**
4. ストックにない新しい言い回しを積極的に生み出すことも推奨する
