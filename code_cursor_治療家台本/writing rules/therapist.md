# 治療家向けルール

---

## この動画の前提

| 項目 | 内容 |
|------|------|
| **話者** | 廣瀬医師 |
| **視聴者** | 整体院・整骨院・鍼灸院などを運営している院長・スタッフ |
| **加盟状況** | すでにコンサルに加盟済みのメンバー。入会勧誘は不要 |
| **目的** | 医学的な知識を正しく理解し、自院の診療・説明・売上に活かす |
| **トーン** | 医師として同じ現場のプロに語りかける。上から目線ではなく、仲間として |

---

## 1動画の基本構造：1つの深い学び＋1つの実践

**1動画で扱うテーマは1つ。** 広く浅くではなく、1つのテーマを深く掘り下げ、それに対応する具体的な実践を1つ渡す。

| 要素 | 内容 |
|------|------|
| **深い学び（1つ）** | テーマの本質・仕組み・なぜそうなるのかを、腑に落ちるレベルまで丁寧に解説する |
| **実践（1つ）** | 視聴後すぐに自院で実行できる具体的なアクションまたは改善案を1つ渡す |

### 実践の条件
- **明日からできる**具体性があること（「意識する」ではなく行動レベルまで落とし込む）
- 院のオペレーション・患者トーク・導線・数字のいずれかに直接つながること
- 1つに絞ること（複数渡すと何もやらない）

### NG
- 複数の学びと複数の実践を詰め込む → 散漫になり何も刺さらない
- 学びだけで実践なしに終わる → 「で、何をすればいいの？」で終わる
- 実践が抽象的すぎる → 「患者との関係を大切にしましょう」レベルはNG

## 感情設計：「入って良かった」を積み上げる

視聴者はすでに加盟済みのメンバー。台本全体でこの3つの感情を引き出す。

| 目指す感情 | 具体的なイメージ |
|-----------|----------------|
| **「入って良かった」** | 加盟した判断が正しかったという確信 |
| **「売上が改善できる」** | 今日学んだことが院の数字に直結する手応え |
| **「ここで学ぶことへの感動」** | FANTSでしか得られない知識・視点に触れる喜び |

### 設計原則
1. **知識＋すぐ使える行動をセットで渡す** → 「学んだ甲斐があった」の充実感につながる
2. **「あなたの院が変わる」を言い切る** → 「この知識を持つ治療家は少ない」「患者への説明の説得力が段違いです」など、加盟メンバーとしての優位性を折に触れて伝える
3. **成果イメージで締める** → 「この内容を使えば〇〇が変わります」など、未来の成功場面を見せて終わる

### NG
- 「まず入会を検討してください」的な誘導（すでに加盟済み）
- 知識の羅列だけで感情的な充実感がない台本
- 「難しいですが頑張ってください」のような突き放した締め

## 測定会との接続

測定会は加盟院にとって**最大のバリューポイント**であり、院の売上に直結するイベント。動画で伝える医学知識は、日々の診療に活きると同時に、測定会での患者説明・成約力の土台になる。

### 測定会とは
- 各治療院が既存顧客または新規顧客を集めて、**廣瀬医師と共同で開催**するイベント
- 治療家単体では実施できない内容であり、医師である廣瀬の参加が不可欠
- 加盟していることで初めて実現できる仕組み＝コンサル加盟の最大の価値

| 原則 | 内容 |
|------|------|
| **日々の診療 → 測定会の二段構え** | 動画の学びはまず日々の診療で使えるが、その延長線上に測定会での成果がある |
| **テーマとの関連度で判断** | テーマが測定会と関連する場合は「この知識が測定会でどう活きるか」を台本内で自然に触れる |
| **押し付けない** | 測定会の話題は自然な文脈で出す。無関係なテーマに無理に絡めない |

### NG：恩着せ・押し売り表現
「入って良かった」は**内容の質で自然に感じてもらうもの**。台本側から加盟の価値を押し付けたり、感謝を求めるような表現は絶対に使わない。

| NG | なぜダメか |
|-----|----------|
| 「加盟しているからこそ聞ける話です」 | 恩着せがましい |
| 「この知識は普通は手に入りません」 | 優越感の押し売り |
| 「だから加盟して正解だったんです」 | 感謝の強要 |

価値は内容で示す。言葉で「ありがたいでしょ」とは言わない。

## ゴール（各動画で達成すること）

- 解説内容をしっかり理解できる（医学的・臨床的な中身を正確に把握）
- 自院に具体として落とし込める（明日から実践できるレベルまで具体化）
- テーマに応じて、測定会での活用イメージが持てる

## 言語レベル：新卒社会人基準

整体師・柔道整復師・鍼灸師は、医学的な生化学・ホルモン学・内科学を深く学んでいないケースが多い。**新卒の社会人が聞いてもわかる表現**を基本とする。

- 1文に情報を詰め込まない（1文1情報）
- 難しい言い回しより「つまり〇〇ということです」と言い切る形を優先

### NG例 → OK例

| NG（難しすぎる） | OK（新卒社会人でもわかる） |
|----------------|----------------------|
| 性腺刺激ホルモン放出ホルモンの分泌抑制により… | 脳からの「ホルモンを出せ」という命令が減るため… |
| 負のフィードバック機構が破綻し… | 体のバランスを保つ仕組みが崩れてしまうんですね |
| 炎症性サイトカインの過剰産生 | 体の中で炎症を引き起こす物質が必要以上に増えてしまう |

## 専門用語の扱い：核心用語だけ台本内で解説する

専門用語は使ってよい。ただし**台本内で解説を入れるのは、その用語を理解していないと動画全体の理解が深まらない場合に限る**。

| 判断 | 条件 |
|------|------|
| **解説する** | 動画のメインテーマ・結論・核心に直結している用語 |
| **解説しない** | 補足的・周辺的な用語で、知らなくても動画の流れを理解できる |

### 解説の入れ方
- 「〇〇——つまり〜〜のことです」
- 「〇〇というのは、簡単に言うと〜〜です」
- 比喩を使う（「車のエンジンに例えると〜」等）

## 動画テンプレ（医学知識解説動画）

治療家向け学習動画の標準構成。1動画＝1テーマ＋1実践の原則に基づく。

### 0:00-0:20　共感フック＋テーマ提示＋今日の約束
- 視聴者の痛み・悩みから入る（「〇〇していませんか？」「〇〇で終わっていませんか？」）
- その直後にテーマと「この動画で得られる結果」を言い切る
- 目次の列挙や概要説明で始めない

### 0:20-1:00　なぜこれを知る必要があるのか（院への直接メリット）
- この知識が自院の患者説明・成約・信頼にどう直結するかを伝える
- 「これを知っているかどうかで、患者さんへの説明の深みが変わります」

### 1:00-4:00　本題：仕組みと本質の解説
- テーマの「なぜ？どうして？」を掘り下げる
- 新卒社会人でもわかる言葉＋核心用語のみ解説
- 比喩・具体例を活用して腑に落とす

### 4:00-6:00　証拠・実績（具体的な変化の提示）
- 「理論はわかった。で、本当に成果が出るの？」に答えるパート
- 素材テキストに数字・事例があればそのまま使う
- なければナレーションに `【▶ 実績：ここに具体的な数字・事例を追記】` の穴埋めを置き、ユーザーが後から加筆できるようにする

### 6:00-8:00　臨床への橋渡し（実際の患者像・ケース）
- 「実際にこういう患者さんが来たとき、この知識がどう活きるか」
- 患者に何と説明するか、どう対応するかを具体的に示す

### 8:00-9:00　実践アクション（1つだけ）
- 「明日から院でこれをやってください」という具体的なアクション1つ
- 院のオペレーション・トーク・導線・数字のいずれかに直結するもの

### 9:00-10:00　まとめ＋成果イメージで締める
- 今日の学びを1〜2文で復唱
- 「この内容を使えば〇〇が変わります」で締める
- 次の動画・学びへの期待感を添えてもよい

---

## ツール・仕組みの具体性（必須）

- ツール名・プラットフォーム名・仕組みの名前を初出で使う際は、**それが物理的に何であるか（動画アプリ／冊子／院内スライド／測定機器 等）を一言で添える**
- 「KENTERAで学んでもらいます」ではなく「KENTERAという動画学習アプリで学んでもらいます」のように、視聴者が頭の中にイメージを持てる状態にする
- 抽象的な名前だけで話を進めない

---

## 用語・表現

- 医学的・臨床的な正確性を最優先にする
- 専門用語は使ってよい（核心用語のみ解説、周辺用語は解説不要）
- 簡潔で要点を押さえた説明を心がける
