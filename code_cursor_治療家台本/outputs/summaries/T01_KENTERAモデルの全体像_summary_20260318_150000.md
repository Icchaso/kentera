【T01：KENTERAモデルの全体像 ── 売り込みゼロで院の数字が変わる仕組み】

〜知っておくと読みやすい用語〜
・KENTERA → 患者教育のための動画学習アプリ。患者さんが健康の基礎知識を動画で段階的に学べる仕組み
・オリゴスキャン → 手のひらに光を当てて、体内のミネラルバランスや有害金属の蓄積状態を測定する機器。非侵襲で短時間に結果が出る
・バックエンド（BE） → 集客・入口の施術に対して、本命となる高単価メニューやサービスのこと
・LTV（ライフタイムバリュー） → 1人の患者さんが長期的に院にもたらす価値の総額。継続率・紹介率が上がるほど高くなる
・有害金属 → 水銀・カドミウム・鉛・アルミニウムなど、体内に蓄積すると健康に悪影響を及ぼす金属。日本人の慢性疾患の60〜70%に関与しているとされる
・患者導線 → 患者さんが初来院から施術・提案・継続に至るまでの一連の流れ。導線の設計が成約率や継続率に直結する

〜この動画でわかること〜
「教育→測定→バックエンド→LTV」の4ステップを順番通りに進めるだけで、なぜ売り込みに頼らず成約・継続・紹介が自然に生まれるのかがスッキリわかります。


【今日の3つのポイント】

【1. 提案の「前」に教育と測定を入れるだけで結果が変わる】
・従来の「説明→提案→断られる」パターンは、先生の提案力ではなく患者さん側の準備不足が原因
・KENTERAモデルでは「教育→測定→納得→提案」の順番に変えることで、同じ提案でも受け取り方がまったく変わる
（例）基礎工事をせずに家を建てるようなもの。土台ができていなければ、どんなに良い材料を使っても上物は安定しない。提案の前に教育と測定で「土台」を作ることが鍵
・何を提案するかではなく、提案の前に何があるかで結果が決まる

【2. KENTERA（動画学習アプリ）と測定で「納得」を仕組み化する】
・KENTERAで患者さんが事前に体の仕組みや基礎知識を学ぶことで、先生の説明負担が大幅に減る
・オリゴスキャンで自分の数値を確認した瞬間、学んだ知識が一気に「自分の課題」に変わる
（例）「有害金属が慢性疾患に関係する」と知識で知っているのと、「自分の体にカドミウムが蓄積している」と数値で見るのでは、受け取り方がまったく違う
・有害金属は日本人の慢性疾患の60〜70%に関与。多くの患者さんに所見が出るため「異常なし」で終わらない
・教育が先にあるからこそ、数値を見たときに「これが自分の問題だ」とつながる

【3. 納得した患者さんは継続し、紹介が自然に生まれる】
・教育と測定で準備ができた状態で提案されるバックエンドは「売り込み」ではなく「自分に必要な改善策」として受け取られる
（例）いつ水が必要かわからない状態で「この肥料を買いましょう」と言われても迷う。でも土壌の状態がわかっていれば、必要なものを自分から選べる
・自分で選んだ患者さんは継続しやすく、良い体験が紹介にもつながる
・院の説明工数も減り、売上と患者満足の両方が長期的に安定する


【今日からやること】

・今日 → 自院の現在の患者導線を紙に書き出す。初診から継続までの流れを可視化するだけでOK
・3日以内 → 書き出した導線の中で「教育が入っていない箇所」「測定で変わりそうな箇所」に印をつけてみる
・1週間後 → 次の動画T02（オリゴスキャン解説）を見て、測定ステップへの理解を深める。全体像を踏まえた上で見ると、測定の位置づけがさらにクリアになる


〜ひとことメッセージ〜
何を提案するかではなく、提案の前に何を整えるか。それだけで、院の数字も患者さんの反応も変わっていきます。
