# T01：KENTERAモデルの全体像 ── 売り込みゼロで院の数字が変わる仕組み

## 1. 「説明疲れ」していませんか？

患者さんに一生懸命説明しても、安いメニューしか選ばれない。単発で終わる。そんな状況を変える「KENTERAモデル」の全体像をお伝えします。

- 売り込みなしで成約・継続・紹介が生まれる順番がわかる
- この動画1本で、院の導線設計の土台が見える

画像：治療院、問診風景、悩む院長

## 2. なぜ今のやり方では限界があるのか

従来の「説明→提案→断られる」パターンは、先生の提案力ではなく患者さん側の準備不足が原因。提案の「前」に何を置くかで結果が変わります。

- 従来：説明 → 提案 → 「ちょっと考えます」
- KENTERAモデル：教育 → 測定 → 納得 → 提案
- 違いは「提案の前に何があるか」

画像：2つのフロー対比図、従来パターンと新パターンの比較

## 3. 結論：4つのステップを順番通りに進める

教育→測定→バックエンド成約→LTV向上。この4ステップを順番通りに進めるだけで、患者さんの納得が先にでき、成約も継続も自然に生まれます。

- ① 教育（KENTERA） → ② 測定（オリゴスキャン） → ③ バックエンド成約 → ④ LTV向上
- ポイントは「いきなり提案しない」こと
- 教育と測定で納得を先に作る

画像：4ステップの矢印フロー図、番号付き、シンプルな図解

## 4. ステップ1：教育（KENTERA動画学習アプリ）

KENTERAという動画学習アプリで、患者さんが体の仕組みや健康の基礎を事前に学びます。先生の説明負担を仕組みで軽減する装置です。

- 患者さんが「なぜ必要か」を理解した状態で来院する
- 先生は「それを踏まえて、あなたの場合は」から話を始められる
- 教育を仕組み化し、毎回ゼロから説明する工数がなくなる

画像：スマホ、動画視聴、患者が学ぶ、アプリ画面

## 5. ステップ2：測定（オリゴスキャン）

オリゴスキャンという測定機器で体内のミネラルバランスと有害金属の蓄積状態を実測。教育で得た知識が「自分の課題」に変わる瞬間です。

- 自分の数値を目で見ることで「知識」が「納得」に変わる
- 有害金属は日本人の慢性疾患の60〜70%に関与
- 「異常なし」で終わらず、具体的な改善のきっかけが見つかる

画像：測定機器、データグラフ、ミネラル分析

## 6. ステップ3→4：バックエンド成約からLTV向上へ

教育と測定を経た患者さんへの提案は「売り込み」ではなく「自分に必要な改善策」として受け取られます。納得して選んだ患者さんは継続しやすく、紹介にもつながります。

- 成約率・満足度・継続率がまとめて変わる
- 院の説明工数が減り、施術に集中できる
- 【▶ 実績：ここに導入院の具体的な数字・事例を追記】

画像：信頼、長期的成長グラフ、リピート、紹介

## 7. 今日の実践アクション

自院の現在の患者導線を紙に書き出してみてください。「教育が入っていない箇所」「測定で変わりそうな箇所」を見つけることが、モデル導入の最初の一歩です。

- 初診から継続までの流れを可視化する
- 4ステップのどこが欠けているか確認する
- 現状把握さえできれば、改善ポイントは自然に見えてくる

画像：ノート、フロー図作成、チェックリスト

## 8. まとめ ── 次の動画への接続

教育で土台を作り、測定で納得を作り、バックエンドで必要な提案をし、その結果として継続と紹介が生まれる。この仕組みを院の導線に組み込めるかどうかで、数字は大きく変わります。

- 次回はオリゴスキャンの仕組みを詳しく解説
- 今日の全体像を踏まえて見ると、測定の位置づけがさらにクリアになる

画像：まとめ、ステップ全体図、次回予告
