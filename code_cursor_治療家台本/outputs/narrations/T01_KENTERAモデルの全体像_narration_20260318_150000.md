【スライド1】
患者さんに一生懸命説明しても、安いメニューしか選ばれない。継続にもつながらない。そんな「説明疲れ」、していませんか？こんにちは、廣瀬です。今日はその状況を変えるKENTERAモデルの全体像をお伝えします。見終わったあと、売り込みに頼らなくても成約・継続・紹介が自然に生まれる仕組みの全体像がクリアに見えるようになります。

【スライド2】
では、なぜ多くの院で同じ悩みが起きるのか。少し整理してみましょう。従来の多くの院では、先生が丁寧に説明して「こういうメニューもありますよ」と提案しても、患者さんは「ちょっと考えます」で終わってしまう。説明して、提案して、断られる。このパターンですね。でもこれ、先生の提案力の問題じゃないんですよ。患者さん側の準備が、まだ整っていなかっただけなんです。KENTERAモデルでは、提案の前に「教育」と「測定」という2つのステップが入ります。この2つがあるかないかで、同じ提案でも患者さんの受け取り方がまったく変わる。つまり、何を提案するかではなく、提案の前に何があるかで結果が決まるということなんです。ここが今日いちばんお伝えしたいポイントです。

【スライド3】
では、その全体像はどうなっているのか。結論はこの4つのステップです。教育、測定、バックエンド成約、LTV向上。ポイントは「いきなり提案しない」ということ。教育と測定で患者さんの納得を先に作っておく。その上で提案するから、成約にも継続にも自然につながっていく。これがこのモデルの骨格です。では、ここから一つずつ見ていきましょう。

【スライド4】
最初のステップは教育です。ここで使うのがKENTERAという動画学習アプリですね。患者さんがこのアプリを使って、体をどういう順番で見ればいいのか、よくある不調がなぜ起きるのか、何から整えればいいのかを段階的に学んでいきます。たとえば、僕が測定会でお伝えしている内容がありますよね。「健康は肉体、精神、社会の3つでできている」とか、「体はDNAという設計図、酵素という職人、栄養素という建材で成り立っている」とか、「慢性疾患に最も関わるのはミネラルと有害金属だ」。こうした基礎知識を、患者さんが来院前にアプリで学んでくれるんです。だから先生は毎回ゼロから全部説明しなくてよくなる。「それを踏まえて、あなたの場合はこうですよ」という話から入れるようになります。KENTERAは患者教育の場であると同時に、院にとっては教育そのものを仕組み化できる装置なんですね。患者さんの理解が先に進んでいるから、「とりあえず通う」ではなく「なぜ自分にこれが必要なのか」をわかった上で来院してくれるようになる。これが次のステップに大きく効いてきます。

【スライド5】
ここからが面白いところなんですが、知識が入っただけでは「ふーん、なるほどね」で止まりやすいんですよ。知識としてはわかっていても、まだ自分ごとにはなっていない。自分ごとになっていない状態で「こういう施術がありますよ」と提案しても、やっぱり響きにくい。そこで測定を入れます。測定会ではオリゴスキャンという機器を使って、体内のミネラルバランスと有害金属の蓄積状態を実際に測ります。患者さんが自分の数値を目の前で見た瞬間、KENTERAで学んだ知識が「自分の課題」に一気に変わるんですよ。「有害金属が慢性疾患に関係する」と知識で知っているのと、「自分の体にカドミウムが溜まっている」と数値で見るのでは、受け取り方がまるで違いますよね。ちなみに有害金属は日本人の慢性疾患の60%から70%に関与しているとされていて、多くの患者さんに何かしらの所見が出ます。「異常なし」で終わらず、具体的な改善のきっかけが見つかる検査なんですね。教育が先にあるからこそ、数値を見たときに「ああ、これが自分の問題なんだ」とつながるわけです。オリゴスキャンの仕組みや読み方については、次のT02の動画で詳しくお話しします。

【スライド6】
ここまでの2ステップを経た患者さんへの提案は、「売り込まれている」ではなく「自分に必要な改善策を教えてもらっている」という感覚になります。水やりで例えてみましょう。いつ、どのくらいの水が必要かわかっていない状態で「この肥料を買いましょう」と言われても、迷いますよね。でも、土壌の状態を調べて「ここが足りていない」とわかっていたら、必要なものを自分から選べる。それと同じなんです。だから成約率だけでなく、満足度も上がりやすい。自分で納得して選んでいるから、継続もしやすい。さらに、いい体験をしている患者さんは周りの人にも「あそこの院はちゃんと説明してくれるよ」と自然に伝えてくれます。紹介が自然発生するんですね。院側にもメリットがあります。毎回ゼロから同じ説明を繰り返さなくてよくなるので、工数が減る分、先生は施術や患者さんとの対話に集中できるようになる。実際にこの順番を導入した院では、数字として見える変化が出ています。【▶ 実績：ここに具体的な数字・事例を追記】。このモデルは一回売って終わりの仕組みではありません。院の売上と患者さんの満足を、長期にわたって安定させていく。そこがこの仕組みの本質です。

【スライド7】
では今日の実践アクション、1つお伝えしますね。自院の現在の患者導線を、一度紙に書き出してみてください。初診の患者さんが来てから、どんな順番で説明し、どう提案し、どうやって継続につなげているか。この流れを可視化するだけで、「ここに教育が入っていないな」「ここで測定があれば反応が変わりそうだな」というのが見えてきます。全部を一度に変えなくていいです。まずは現状を把握すること。それがこのモデルを自院に落とし込む最初の一歩になります。

【スライド8】
最後にまとめますね。教育で土台を作り、測定で納得を作り、バックエンドで必要な提案をし、その結果として継続と紹介が生まれる。この仕組みを院の導線に組み込めるかどうかで、数字は大きく変わっていきます。次の動画では、このモデルの中核を担うオリゴスキャンの仕組みについて詳しくお話しします。今日の全体像を頭に入れた上で見ると、測定がなぜここまで重要なのかがさらにクリアになりますので、ぜひ続けて見てみてください。今日の内容が、院の導線を見直すきっかけになれば嬉しいです。また次の動画でお会いしましょう。
