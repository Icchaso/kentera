# KENTERA 解説動画 台本生成

医療データを含む元テキストから、患者向けまたは治療家向けの解説動画台本を生成します。

**Cursor 内の Claude Opus 4.6** で、ルールに従って編集・生成し、**編集後台本/** に自動保存されます。

---

## 使い方（3ステップ）

1. **Cursor のチャットに元テキストを貼り付ける**
2. **「患者向けで台本作って」または「治療家向けで台本作って」と依頼する**
3. **Opus 4.6 を選択** → ルール通りに台本が作られ、`編集後台本/` に自動保存される

例：
```
（元テキストを貼り付け）

患者向けで台本作って
```

---

## フォルダ構成

```
KENTERA台本/
├── 元の素材/             # 元テキスト（参照・貼り付け用）
│   └── sample_input.txt
├── 編集後台本/           # 生成された台本（自動保存先）
├── rules/               # ルール・基準
│   ├── common.md       # 共通ルール
│   ├── format.md       # 台本フォーマット
│   ├── patient.md      # 患者向け基準
│   └── therapist.md    # 治療家向け基準
├── .cursor/rules/      # Cursor 用（自動保存の指示）
└── README.md
```
