# 契約院管理

## 管理対象

コンサル契約中の治療院を院別に管理する。

---

## フォルダ構成

- `院別/` ← 各院のページを作成
  - ファイル名例：`院別/〇〇整骨院.md`

---

## 院別ページの記載項目（テンプレート）

[[../08_テンプレート/契約院ページテンプレート]] を使用

---

## KPI管理

| 指標 | 目標 | 現状 |
|------|------|------|
| 契約院数 | - | - |
| 測定会開催回数 / 月 | - | - |
| バックエンド成約率 | - | - |
| 継続率（3ヶ月） | - | - |

---

## 継続のポイント

- 導入後30日設計を実行できているか確認
- 月次KPIを一緒に見る
- 測定会の開催頻度を維持させる
