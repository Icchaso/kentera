# IDMS Vault — Claude Code ルール

## 組織概要
- 一般社団法人難治性疾患医学会（IDMS） — 代表理事 廣瀨和則
- 東洋医学×西洋医学の融合をテーマとした医学会
- KENTERA（医療コンサル事業）はIDMS配下の事業

## 事業構成
- **IDMS（医学会本体）** — 組織運営・公式HP・ネットワーク
- **KENTERA（コンサル事業）** — toC（患者向け）× toB（治療家向け）の2軸。FANTS・LINE・YouTube を活用した動画コンテンツ配信

## GitHub
- リポジトリ: Icchaso/idms

## フォルダ構成
- 00_ダッシュボード/　　　　← HOME・INDEX
- 01_IDMS（医学会本体）/　　← 組織概要・公式HP・ネットワーク
- 02_KENTERA（コンサル事業）/← 動画台本・LINE設計・測定会・契約院・営業
- 03_支援事例・実績/　　　　← 成果事例
- 04_ミーティング/　　　　　← 議事録
- 05_テンプレート/　　　　　← ミーティング・契約院ページのテンプレート
- 06_ClaudeCode活用アイデア/ ← Claude連携アイデア

## 動画シリーズ体系（KENTERA）
### toC（患者向け）02_KENTERA（コンサル事業）/toC_患者向け/FANTS/
- A: 導入シリーズ
- B: 基礎9ステップ（B01〜B09、全台本完成済み）
- C: 症例別シリーズ（C01〜C16）

### toB（治療家向け）02_KENTERA（コンサル事業）/toB_治療家向け/FANTS/
- T: モデル理解
- U: 院内FANTS導入
- V: 測定会の回し方
- W: 契約・継続

## ステータス管理
動画制作の進捗は `02_KENTERA（コンサル事業）/動画制作マスタートラッカー.md` で一元管理:
- 🔴 = 優先度高・未着手
- 🟡 = 台本完成（撮影待ち）
- 🔵 = 完成（編集済み）

## 命名規則
- 動画台本: `[シリーズ記号][番号]_タイトル_台本.md`（例: B03_高血糖総論_台本.md）
- 議事録: `YYYY-MM-DD_タイトル.md`
- インデックス: `[カテゴリ名]インデックス.md`
- 参考素材: `タイトル（YouTube書き起こし）.md`
- フォルダ: `NN_カテゴリ名`

## 内部リンク
- `[[ファイル名]]` または `[[パス/ファイル名]]` 形式
- インデックスファイルからコンテンツへリンク
- `00_ダッシュボード/HOME.md` がVault全体のダッシュボード

## テンプレート（05_テンプレート/）
- ミーティングテンプレート.md — 議事録の雛形
- 契約院ページテンプレート.md — 院別管理ページの雛形

## 注意事項
- 患者の個人情報を含むノートは作成しないこと
- 動画台本の著作権は KENTERA/IDMS に帰属
- 参考素材（YouTube書き起こし）は社内利用のみ
- 既存の台本ファイルは構成を壊さないよう注意すること
