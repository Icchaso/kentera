# KENTERA Vault — Claude Code ルール

## 事業概要
- 医療コンサル・栄養学教育事業（KENTERA）
- toC（患者向け）× toB（治療家向け）の2軸
- FANTS・LINE・YouTube を活用した動画コンテンツ配信
- GitHub: Icchaso/kentera

## フォルダ構成
- 00_事業設計/　　　　　　← ビジョン・事業モデル・重要判断ログ
- 01_toC_患者向け/　　　　← 患者向け動画台本（FANTS）・LINE設計
- 02_toB_治療家向け/　　　← 治療家向け動画台本（FANTS）・面談設計・参考素材
- 03_測定会/　　　　　　　← 測定会イベント管理
- 04_契約院管理/　　　　　← 院別の進捗・KPI管理
- 05_営業・新規獲得/　　　← 営業活動
- 06_支援事例・実績/　　　← 成果事例
- 07_ミーティング/　　　　← 議事録
- 08_テンプレート/　　　　← ミーティング・契約院ページのテンプレート
- 09_ClaudeCode活用アイデア/ ← Claude連携アイデア

## 動画シリーズ体系
### toC（患者向け）01_toC_患者向け/FANTS/
- A: 導入シリーズ
- B: 基礎9ステップ（B01〜B09、全台本完成済み）
- C: 症例別シリーズ（C01〜C16）

### toB（治療家向け）02_toB_治療家向け/FANTS/
- T: モデル理解
- U: 院内FANTS導入
- V: 測定会の回し方
- W: 契約・継続

## ステータス管理
動画制作の進捗は `動画制作マスタートラッカー.md` で一元管理:
- 🔴 = 優先度高・未着手
- 🟡 = 台本完成（撮影待ち）
- 🔵 = 完成（編集済み）

## 命名規則
- 動画台本: `[シリーズ記号][番号]_タイトル_台本.md`（例: B03_高血糖総論_台本.md）
- 議事録: `YYYY-MM-DD_タイトル.md`
- インデックス: `[カテゴリ名]インデックス.md`
- 参考素材: `タイトル（YouTube書き起こし）.md`
- フォルダ: `NN_カテゴリ名`

## 内部リンク
- `[[ファイル名]]` または `[[パス/ファイル名]]` 形式
- インデックスファイルからコンテンツへリンク
- `HOME.md` がVault全体のダッシュボード

## テンプレート（08_テンプレート/）
- ミーティングテンプレート.md — 議事録の雛形
- 契約院ページテンプレート.md — 院別管理ページの雛形

## 注意事項
- 患者の個人情報を含むノートは作成しないこと
- 動画台本の著作権は KENTERA に帰属
- 参考素材（YouTube書き起こし）は社内利用のみ
- 既存の台本ファイルは構成を壊さないよう注意すること
